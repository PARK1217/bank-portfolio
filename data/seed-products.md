# 시드 상품 라인업 (DB 세션 참고용)

> **목적**: `db/02_seed.sql` 작성 시 `public."PRODUCT"` + 부속 테이블에 INSERT 할 상품 라인업.
> **명칭 정책**: 시중은행 일반 상품 패턴을 *가공*. 실명·실상품 그대로 X.
> **본행 코드**: `020` (가상은행, README 기준 가정).
> **상품 ID**: 100번대부터 부여 — DB 세션이 충돌 없으면 그대로 사용.

---

## 0. 컬럼 매핑 요약 (PRODUCT 테이블)

| 본 문서 필드 | DB 컬럼 | 비고 |
|---|---|---|
| 상품ID | `PRODUCT_ID` (smallint) | |
| 상품명 | `PRODUCT_NAME` varchar(80) | |
| 유형 | `PRODUCT_TYPE_CD` varchar(8) | SAVING / DEPOSIT / INSTALLMENT / LOAN |
| 특판여부 | `SPECIAL_YN` char(1) | Y/N |
| 중도해지가능 | `EARLY_CLOSE_YN` char(1) | |
| 자동연장 | `EXTEND_YN` char(1) | 정기예금만 |
| 가입연령 | `MIN_AGE` / `MAX_AGE` | |
| 만기정책 | `MATURITY_POLICY_CD` | NORMAL / EXTEND / REPAY |
| 대상고객 | `TARGET_CUSTOMER_CD` | ALL / INDIV / CORP / MINOR / YOUTH / SENIOR / FOREIGN |
| 최소·최대 금액 | `MIN_AMOUNT` / `MAX_AMOUNT` (bigint) | 원 |
| 월납입 최소·최대 | `MIN_MONTHLY_AMT` / `MAX_MONTHLY_AMT` (integer) | 적금만 |
| 이자지급주기 | `INTEREST_CYCLE_CD` | MONTHLY / QUARTERLY / MATURITY |
| 출시일·판매시작·종료 | `LAUNCH_DATE` / `SALE_START_DATE` / `SALE_END_DATE` | YYYYMMDD |

**부속 테이블** (각 상품에 1:N):
- `PRODUCT_PERIOD` — 가입 가능 기간 + 기간별 기본 금리
- `PRODUCT_RATE_POLICY` — 금액 구간별 금리
- `PRODUCT_BONUS_CONDITION` — 우대 조건 + 가산 금리
- `PRODUCT_TERMS_MAPPING` — 적용 약관 (terms-corpus 와 연계)

## 1. SAVING (입출금) — 5종

### P-101 본행 자유입출금 통장
- 유형 SAVING / 일반 / 대상 ALL / 가입연령 14~99
- 한도: 1일 출금 5,000만원 / 1회 이체 1억
- 기본 금리: 연 0.10% (월 이자)
- 적용 약관: 여신거래기본 / 전자금융 / 개인정보 (필수 3)
- 시나리오 사용: **박철수 110-001-100001**, **김영희**, **한도현**

### P-102 급여이체 우대 통장
- 유형 SAVING / 일반 / 대상 INDIV / 가입연령 19~70
- 우대 금리: 급여이체 시 연 1.50% (50만원까지)
- 우대 조건:
  - SALARY: 월 100만원 이상 급여이체 3개월 연속 → +1.40%p
  - CARD: 월 30만원 카드 사용 → +0.30%p
- 시나리오 사용: **박철수 (급여이체)**, **송은행 (우대 모든 항목 충족)**

### P-103 마이너스통장 한도형 (수신·대출 일원화)
- 유형 SAVING / 일반 / 대상 INDIV / 가입연령 19~65 / 한도 500만~1억
- ⭐ **계좌-계약 일원화**: 1계좌 = 수신 + 대출. 잔액 음수 = 대출 사용
- 기본 금리: 입금 시 0.10% / 출금 시 6.5%p (마이너스 대출이자)
- 우대 조건:
  - CREDIT_GRADE_HIGH: 1~3등급 → -0.50%p
  - SALARY_TRANSFER: 급여이체 → -0.30%p
- 시나리오 사용: **김미선 110-006-100001** (한도 5천만, 잔액 -2,300만)

### P-104 외화 자유입출금 통장 (USD/EUR/JPY)
- 유형 SAVING / 일반 / 대상 ALL (외국인 가능) / 가입연령 14~99
- 통화: USD / EUR / JPY / CNY / GBP (5종)
- 기본 금리: 통화별·시기별 변동 (USD 약 4.5% 기준)
- 시나리오 사용: **John Smith 110-008-100001** (USD)

### P-105 법인 당좌예금
- 유형 SAVING / 일반 / 대상 CORP / 가입연령 X (법인)
- 한도: 사업자 신용평가 기반
- 기본 금리: 0.05%
- 시나리오 사용: **두리테크 110-009-100001**

## 6. DB 세션 작업 안내

1. 위 25개 상품을 `public."PRODUCT"` 에 INSERT (ID 100~408 유지 권장 — 충돌 시 조정)
2. 각 상품의 `PRODUCT_PERIOD` 항목 INSERT (가입 기간별 금리)
3. 각 상품의 `PRODUCT_BONUS_CONDITION` INSERT (우대 조건 — 위 명시된 것)
4. `PRODUCT_RATE_POLICY` 는 단순 상품(P-101 등)은 단일 행, 복잡 상품은 다단계 행
5. `PRODUCT_TERMS_MAPPING` 은 `data/seed-terms/` 약관 ID 와 매핑 (Phase C 산출물 대기)
6. **사잇돌·새희망홀씨**는 정책 대출이라 별도 `LOAN_POLICY_CD` 컬럼 활용 (있으면)
7. **외국인 상품**은 `TARGET_CUSTOMER_CD = FOREIGN` 으로 필터되어 외국인고객만 노출
