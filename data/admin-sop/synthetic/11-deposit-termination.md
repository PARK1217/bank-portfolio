# 예금·적금 중도해지 SOP (직원 고객응대 매뉴얼)

> **AUDIENCE**: ADMIN  **CATEGORY**: BANKING_OPS  **BASIS**: 예금자보호법, 은행연합회 정기예금 표준약관, 적금 표준약관, 금소법 §17·§19

## 1. 해지 가능 여부 1차 판정 (창구·콜센터 1분 안내)

### 1-1. 해지 가능 케이스
| 상품 | 해지 가능 시점 | 비고 |
|---|---|---|
| 자유입출금 (수시입출식) | 언제든 가능 | 잔액 0원 + 자동이체 미사용 시 |
| 정기예금 | 만기 전 중도해지 가능 | 중도해지 이자율 적용 |
| 정기적금 | 만기 전 중도해지 가능 | 회차별 납입금 + 중도해지 이자율 |
| 외화예금 | 만기 전 중도해지 가능 | 환율 변동 손실 안내 |
| 마이너스 통장 | 잔액 ≤ 0 일 때만 | 한도 사용분 전액 상환 후 |

### 1-2. 해지 불가·제한 케이스
- 담보 설정된 예금 (대출 담보) → 담보 해지 선행
- 압류·가압류 통보 받은 계좌 → 법원 결정 선행
- 휴면 계좌 (LOST/DORMANT) → 본인확인 강화 절차 선행
- 사망 신고된 명의 → 상속 절차 선행
- 미성년 통장 → 친권자 공동 동의 필요

## 2. 청약철회 vs 중도해지 (개념 구분)

### 2-1. 청약철회 (금소법 §19)
- **금소법상 청약철회권은 보장성 보험·대출에만 적용** — **예금·적금에는 적용 X**
- 따라서 7일 전 가입 정기예금 고객이 "청약철회"를 요구해도 **법적 의무 없음**
- 다만 **자유로운 중도해지**는 언제든 가능 (이자율만 중도해지 이자율로 차감)

### 2-2. 중도해지 (정기예금·적금 약관)
- 만기 전 해지 시 약관상 명시된 **중도해지 이자율** 적용
- 가입 후 7일 → 보통 약정금리의 30~50% 수준
- 가입 후 1년 → 약정금리의 50~70%
- 가입 후 만기 임박 → 약정금리의 80~90%

## 3. 고객 응대 표준 문구 (Script)

### 3-1. 가입 7일 후 해지 요청 시 (정기예금)
> "고객님, 정기예금은 가입 후 언제든 해지 가능합니다. 다만 약관에 따라 **약정금리(연 3.5%) 대신 중도해지 이자율(연 1.0%)이 적용**되어 받으실 이자가 줄어듭니다. 7일 경과분 이자는 약 OOO원입니다. 그래도 해지하시겠습니까?"

→ 동의 시 해지 진행. 거부 시 만기 유지 안내.

### 3-2. 담보 설정·압류 등 해지 불가 사유
> "고객님, 본 예금은 [대출 담보 / 법원 압류]가 설정되어 있어 즉시 해지가 어렵습니다. [담보 해지 / 압류 해제] 절차를 먼저 진행해야 하며, 상세 안내를 위해 본부 [여신부 / 법무부]에 연결드리겠습니다."

→ 「해지 보류 안내문」 교부 + 본부 이관 + 처리 기한 안내.

### 3-3. 손해 보전 요구 (불완전판매 주장)
> "고객님, 가입 시 안내드린 약관과 다른 내용이 있었다면 정확히 확인해드리겠습니다. 가입 시점 녹취·약관 동의 화면을 조회 후 [민원 접수서]에 기록해 본부 민원관리부로 이관하겠습니다. 처리 기한은 영업일 14일 이내입니다."

→ 「민원접수서」 작성 + C01 (상품 약관 미흡) 유형 분류 + 본부 이관.

## 4. 처리 절차

### 4-1. 1단계 — 신원·잔액·담보 확인
1. 본인 확인 (신분증 + 비밀번호 또는 OTP)
2. 잔액·이자 발생액 조회
3. 담보·압류 여부 시스템 확인
4. 자동이체 출금계좌 사용 여부 확인 — 사용 중이면 변경 권유

### 4-2. 2단계 — 중도해지 이자 계산·안내
- 대출 처리 모듈 와 유사한 회차 계산 — 일할계산
- 영업일 기준 약정금리 × 보유일수 / 365 × 중도해지 비율

### 4-3. 3단계 — 약관 재확인·동의 징구
- 약관 §15 (중도해지 조항) 화면 노출 + 고객 동의 확인 클릭
- 음성 녹취 진행 — "본 예금을 중도해지합니다. 중도해지 이자율 적용에 동의합니다"

### 4-4. 4단계 — 시스템 처리
1. 예금 계약 상태 해지 전환
2. 원금 + 중도해지 이자 → 본인 명의 다른 계좌 입금
3. 자동이체 사용 중이었으면 자동 정지
4. 해지 확인서 발송 (SMS·이메일)

## 5. 자주 발생하는 분쟁

### 5-1. "직원이 중도해지하면 손해라는 말 안 했다"
- 가입 시점 녹취·약관 동의 화면 캡쳐 보존 5년 → 즉시 조회
- 본인 동의 클릭 기록 (`CUSTOMER_TERMS_AGREE.AGREE_DATETIME`) 확인

### 5-2. "예상 이자보다 적게 받았다"
- 중도해지 이자율은 약관 명시 — §15 인용
- 7일 전 가입 + 약정금리의 30% 적용 결과 정상

### 5-3. "청약철회권으로 환불받겠다"
- **금소법상 예금에는 청약철회권 미적용** 안내 (보장성 보험·대출 한정)
- 다만 자유로운 중도해지는 가능 — 중도해지 이자율 적용

## 6. 시스템 연동
- `예금 계약` 상태 정상 → 해지
- `TRANSACTION` 출금 거래 1건 INSERT (원금+이자)
- `자동이체` ACTIVE 였으면 취소
- `알림` "예금 해지 완료" 자동 발송

## 7. 위반 시 제재
- 약관과 다른 임의 처리 — 금소법 §32 민원 신고 대상
- 중도해지 이자 미적용·과다 적용 — 정산 오류 → 본부 정산부 보고
- 고객 손해 발생 시 — 약관 §16에 의거 보상 검토
