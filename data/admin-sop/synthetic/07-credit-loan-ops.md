# 대출 심사·집행·연체관리 SOP

> **AUDIENCE**: ADMIN  **CATEGORY**: LOAN_OPS  **BASIS**: 여신전문금융업법, 가계대출 종합대책 (DSR/DTI), 대출모범규준

## 1. 신용대출 심사 단계

### 1-1. 1단계 — Pre-screening (자동)
1. 신용평가사 조회 (NICE·KCB) — 신용점수 600점 이상 진행
2. DSR·DTI 자동 계산
   - **DSR (총부채원리금상환비율)**: 모든 대출 원리금 / 연소득 ≤ 40% (규제지역) 또는 50% (비규제)
   - **DTI (총부채상환비율)**: 주택담보대출 + 신용대출 이자 / 연소득 ≤ 50%
3. 회사 재직 6개월 이상 + 4대보험 가입 확인

### 1-2. 2단계 — AI 자동 분류 (시스템)
- 모델: XGBoost (Phase 6 §9.2.6)
- 입력 피처: 신용점수·연소득·기존부채·DSR·재직기간·연체이력
- 출력 SCORE 0~1
  - **≥0.80**: AUTO_APPROVE (자동 승인)
  - **0.40~0.80**: HUMAN_REVIEW (사람 검토 큐)
  - **<0.40**: AUTO_REJECT (자동 거절)

### 1-3. 3단계 — Human Review (회색지대만)
- 본부 여신심사부 담당자 검토
- 추가 서류 요구 가능:
  - 소득금액증명원 (국세청)
  - 재직증명서 (3개월 이내)
  - 사업소득자: 종합소득세 신고서 (3개년)
- 검토 결과: APPROVE·REJECT·CONDITIONAL (조건부 승인)

### 1-4. 4단계 — 약정·실행
- 약정서 전자서명 (PKI 인증)
- 첨부서류 4종 검증 (주민등록증·재직증명·소득증명·가족관계)
- 실행 — `LOAN_EXEC_HISTORY` 적재 + 입금계좌로 자금 이체
- 회차별 상환 스케줄 자동 생성 (원리금균등 EPI)

## 2. DSR 계산식 (가이드라인)
```
DSR = (해당 대출 연간 원리금 + 기존 대출 연간 원리금) / 연소득
```

### 2-1. 산정 시 포함 대출
- 주택담보대출 (원리금)
- 신용대출 (원리금)
- 마이너스통장 (한도의 70% × 연이자율)
- 카드론 (원리금)
- 학자금대출 (원리금, 일부 면제)

### 2-2. 제외 대출
- 전세자금대출 (보증대출 한정)
- 한도 1억 원 이하 미시행 마통

## 3. 연체 관리 단계

### 3-1. 단계별 처리
| 연체 일수 | 단계 | 조치 |
|---|---|---|
| 1~7일 | 단순 연체 | SMS·앱 푸시 안내 |
| 8~30일 | 1차 독촉 | 본인 전화·내용증명 발송 |
| 31~60일 | 2차 독촉 | 본부 채권관리부 이관 |
| 61~90일 | 신용회복 안내 | 한국신용정보원 등록 |
| 91일~ | 채권 매각 검토 | 부실채권 분류·자산건전성 등급 하향 |

### 3-2. 채권 매각 기준
- 90일 이상 연체 + 회수 가능성 낮음
- 자산건전성 분류 "고정" 이하
- 부실채권 매각업체에 양도 (대출원금의 20~40% 회수)

## 4. 연체이자 산정
```
연체이자 = 원금 × (약정금리 + 연체가산금리) × 연체일수 / 365
```
- 연체가산금리: 약정금리 + 3%p (상한 — 대부업법 §8)

## 5. 차주 보호 — 금소법 §17 (대출모범규준)
- 대출 권유 시 적합성·적정성 평가
- 청약철회권 — 14일 이내 무조건 철회 가능 (계약자 의사로)
- 중도상환수수료 — 3년 이내 상환 시 잔액의 1~1.5% (3년 후 면제)

## 6. 채무조정 (워크아웃·개인회생)
- 차주 신청 시 채무조정 협의
- 1순위 조정안 — 이자 감면·만기 연장
- 합의 불성립 시 신용회복위원회 이관
- 개인회생 결정 시 — 법원 절차 준수

## 7. 시스템 연동
- `AI_LOAN_DECISION` — ML 분류 결과 영구화
- `LOAN_APPLICATION` `LOAN_CONTRACT` `LOAN_REPAY_SCHEDULE` 세 단계
- 회색지대(HUMAN_REVIEW) → 관리자 콘솔 `/loans/review-queue` 진입
- 사람 검토 결정 → `HUMAN_DECISION_CD` UPDATE + ADMIN_AUDIT_LOG 자동 적재
