# 시스템 운영·점검·장애 통지 SOP

> **AUDIENCE**: ADMIN  **CATEGORY**: SYSTEM_OPS  **BASIS**: 전자금융감독규정, 금융감독원 IT 운영 가이드

## 1. 시스템 점검 일정

### 1-1. 정기 점검
- **매월 첫째 토요일 02:00~06:00** — 전체 시스템 점검
- 영향 — 모바일뱅킹·인터넷뱅킹·ATM 일부 거래 중단
- 사전 공지 — 7일 전 홈페이지·앱 푸시·SMS

### 1-2. 분기 점검
- 매분기 마지막 일요일 — 결제망 정산·DB 백업·로그 압축
- 영향 — KFTC 자동이체 일시 지연

### 1-3. 긴급 점검
- 보안 패치·핫픽스 — 즉시 적용
- 영향 최소화 — 부분 트래픽 우회 또는 짧은 무중단 배포

## 2. 사전 공지 채널·기한

| 점검 유형 | 공지 기한 | 채널 |
|---|---|---|
| 정기 점검 (월) | D-7 | 홈페이지·앱 푸시·SMS·이메일 |
| 분기 점검 | D-14 | 홈페이지·고객센터 자동 안내 |
| 긴급 점검 | 즉시 | 홈페이지 상단 배너·앱 |
| 외부 결제망 점검 | D-3 (전달받는 즉시) | 홈페이지 |

## 3. 점검 시 임시 조치

### 3-1. 거래 보류
- 진행 중 PENDING 거래 — 점검 종료 후 자동 재시도
- 신규 거래 — "잠시 후 다시 시도해 주세요" 안내

### 3-2. 자동이체 도래
- 점검 시간 도래분 → 점검 종료 후 자동 실행
- 영업일 다음 도래일까지 미실행 — DELAY 상태 적재 (`AUTO_TRANSFER_EXEC.DELAY_REASON_CD='MAINTENANCE'`)

### 3-3. 알림 처리
- 점검 시작 시점 — `NOTIFICATION` SYSTEM 카테고리 일괄 발송
- 점검 종료 시점 — "복구 완료" 알림

## 4. 장애 발생 시 통지 SOP

### 4-1. 1단계 — 인지·1차 대응 (0~10분)
1. 모니터링 시스템 (`EXTERNAL_API_HEALTH` + Phoenix trace + 직원 보고) 알림 수신
2. 본부 IT 운영실 → 즉시 비상대책반 가동
3. 홈페이지·앱 상단 배너 노출 — "현재 일부 거래가 지연되고 있습니다"

### 4-2. 2단계 — 영향 분석 (10~30분)
1. 영향 범위 — 거래 종류·고객 수·금액 추정
2. 「장애 1차 보고서」 작성 — Level 분류 (Level 1~4)
3. 본부 임원 보고

### 4-3. 3단계 — 외부 보고
- **Level 2 이상**: 금감원 IT감독국 통지 (1시간 이내)
- **Level 3 이상**: 한국은행 + 금융위 동시 보고 (즉시)
- **Level 4**: CEO 보고 + 외부 비상통신 가동

### 4-4. 4단계 — 사용자 통지 채널
| 채널 | 시점 | 내용 |
|---|---|---|
| 홈페이지·앱 배너 | 인지 즉시 | 발생 사실 + 진행 상황 |
| 앱 푸시 | 30분 이상 지속 | 영향 범위 + 임시 대응 안내 |
| SMS | 2시간 이상 지속 | 모든 고객 + 진행 상황 |
| 콜센터·영업점 | 즉시 | 「장애 안내문」 비치 |

### 4-5. 5단계 — 복구 후 사후 처리
1. 거래 정합성 점검 — PENDING/FAILED 거래 전수 조회·대조
2. 손해 발생 시 보상 — 약관 §15 의거 검토
3. 「장애 종결 보고서」 — D+14 까지 금감원 제출
4. 재발 방지 대책 — 분기 보고

## 6. 영업시간 vs 비영업시간

### 6-1. 영업시간
- 영업점 — 평일 09:00~16:00 (창구 거래)
- 콜센터 — 24시간 365일
- 인터넷뱅킹 — 24시간 (점검 제외)
- ATM — 24시간

### 6-2. 비영업시간 거래
- 인터넷·모바일 — 정상 (시스템 점검 외)
- 영업점 — 사실상 불가 (긴급 ATM 대응만)
- BOK-Wire — 운영시간(평일 09:00~17:30) 외 거래 거부 (`E_BOK_WIRE_CLOSED`)

## 7. 시스템 점검·장애 이력 공시
- 분기별 점검 횟수·시간·영향 — 홈페이지 공시
- 연간 시스템 가용성 — 99.95% 이상 (목표)
- 미달 시 — 임원 책임 + 개선 계획 공시
