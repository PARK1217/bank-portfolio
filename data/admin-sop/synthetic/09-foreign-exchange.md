# 외환·해외송금 SOP

> **AUDIENCE**: ADMIN  **CATEGORY**: FX  **BASIS**: 외국환거래법, 외국환거래규정

## 1. 외환 거래 유형

### 1-1. 거주자 구분
| 구분 | 정의 | 한도 |
|---|---|---|
| 거주자 | 국내 거주 6개월 이상 내·외국인 | 연 5만 달러 (증빙 무) / 무제한 (증빙 유) |
| 비거주자 | 그 외 | 일반적으로 증빙 필요 |

### 1-2. 거래 목적별
| 목적 | 한도 (1회) | 증빙서류 |
|---|---|---|
| 일반 송금 (증빙 무) | 5,000 USD | 없음 |
| 학비 송금 | 무제한 | 입학허가서·등록금영수증 |
| 의료비 송금 | 무제한 | 진료비영수증·진단서 |
| 해외부동산 취득 | 신고 후 가능 | 매매계약서·소득증빙 |
| 해외증권 투자 | 신고 후 가능 | 투자계약서 |
| 거주자→비거주자 증여 | 연 5만 USD 신고 후 | 증여계약서 |

## 2. 해외송금 절차

### 2-1. 1단계 — 송금 신청
1. 송금자·수취자 정보 입력
2. 송금 목적 선택 (12개 코드 중)
3. 증빙서류 업로드 (필요 시)

### 2-2. 2단계 — 본인확인·신고
- 5,000 USD 이상 — 본인확인 강화 (영상통화·창구 방문)
- 1만 USD 이상 — **국세청 자동 보고** (외국환거래법 §17의2)
- 신고 대상 거래 — 「외국환거래 신고서」 작성 후 한국은행 신고

### 2-3. 3단계 — 환율 적용
- 송금 — TT(전신환) 매도율
- 수취 — TT 매입율
- 자동 환전 — 시장환율 + 스프레드 1~2%

### 2-4. 4단계 — 송금 실행
- 해외 환거래은행 (Correspondent Bank) 경유
- SWIFT MT103 메시지 전송
- 통상 2~3영업일 소요 (특급송금: 당일·익일, 수수료 추가)

### 2-5. 5단계 — 수취 확인
- 수취 은행 입금 확인 회신
- 미입금 시 트레이싱 — SWIFT GPI 실시간 추적
- 7일 이상 미입금 → 환수 절차

## 3. 환전 거래

### 3-1. 영업점 환전
- 1회 1만 달러 이상 — 본인확인 + 자금 출처 진술
- 외화현찰 보유 한도 — 본점 일계 한도 내

### 3-2. 외화 계좌
- USD·JPY·EUR·CNY 등 주요 통화
- 계좌별 통화 단위 — 혼합 보관 불가
- 외화 → 원화 환전: 매입율 적용
- 원화 → 외화 환전: 매도율 적용

## 4. AML 강화 거래 (FATF 권고)

### 4-1. 고위험 국가
- FATF 비협조국 (FATF Public Statement)
- 자금세탁 우려 국가 (한국 금융정보분석원 지정)
- 거래 시 EDD (강화된 본인확인) 의무

### 4-2. PEP (정치적 주요인물)
- 외국 정부 고위공직자·가족·측근
- 송수신 시 사전 본부 컴플라이언스 승인

## 5. CRS·FATCA 보고 (해외계좌 정보 자동 교환)

### 5-1. CRS (OECD 공통보고기준)
- 비거주자 신규 계좌 — 「자기증명서」 (Self-Certification) 의무
- 미국·일본·EU 등 100여 개국 정보 자동 교환

### 5-2. FATCA (미국 해외계좌신고법)
- 미국 시민·영주권자 계좌 — IRS 자동 보고
- W-8BEN·W-9 양식 징구

## 6. 시스템 연동·기록
- `ACCOUNT_TYPE_CD='FOREIGN'` — 외화 계좌
- `TRANSFER_TYPE_CD='KFTC_FX'` — 외환 송금
- `EXTERNAL_API_HEALTH` — SWIFT GPI 상태 모니터링
- 모든 외환 거래 — `ADMIN_AUDIT_LOG` 추가 + 5년 보존
- CRS·FATCA 보고 — 분기별 자동 추출 → 국세청 전송

## 7. 자주 발생하는 문제
- 환율 변동 손실 — 송금 진행 중 환율 변경 시 명시
- 수취 은행 정보 오류 — IBAN·SWIFT 코드 자동 검증
- 송금 목적 코드 오기 — 사후 정정 가능 (영업일 5일 이내)
- 1만 USD 이상 자동 보고 미인지 → 사용자에게 사전 안내 강화
