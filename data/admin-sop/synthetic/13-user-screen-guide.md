# 다온뱅크 사용자 화면 안내 (메뉴 경로·사용법)

> **AUDIENCE**: BOTH  **CATEGORY**: SCREEN_GUIDE  **목적**: 고객·직원이 "어디서·어떻게" 질문에 답하기 위한 화면 경로 가이드

## 1. 메인 진입 (로그인 후)

### 1-1. 대시보드 (홈)
- 경로: 로그인 후 자동 진입 또는 헤더 "대시보드"
- 표시: 총 자산 + 이번 달 수입·지출 + 계좌 카드 목록 + 최근 거래 + 미읽음 알림 + 보유 대출
- 자주 묻는 질문 답변에 우선 안내할 메뉴

### 1-2. 우측 상단 사용자 메뉴 (드롭다운)
- 위치: 헤더 우측 "박철수 님 ▼" 클릭
- 5개 메뉴: 보안 설정 / 상품 카탈로그 / 알림 센터 / 공지·이벤트 / 로그아웃

## 2. 계좌 관련

### 2-1. 계좌 목록
- 경로: 헤더 "계좌" 탭
- 기능: 전체 계좌 카드 보기, 외화 별도 표시, 숨김 계좌 토글

### 2-2. 계좌 상세
- 경로: 계좌 목록 → 카드 클릭
- 기능: 잔액·한도·최근 거래 5건, 통장 인쇄·별명 변경·한도 변경·숨김·분실 신고·해지 5종 액션

### 2-3. 거래 내역
- 경로: 계좌 상세 → "거래 내역" 또는 헤더 "계좌" → 거래 내역
- 기능: 메모·예금주명·계좌번호 검색 + 기간 칩(오늘/1주/1개월/3개월/전체) + 페이지네이션

### 2-4. 통장 출력 (Passbook)
- 경로: 계좌 상세 → "통장"
- 기능: 통장 페이지 형태로 거래 출력

## 3. 이체

### 3-1. 즉시이체
- 경로: 헤더 "이체" → "즉시이체" 또는 dashboard 카드 "이체"
- 단계: 출금계좌 → 입금은행·계좌번호 → verify-account 자동 확인 → 금액·메모 → 확인 → OTP/PIN → 완료
- 빠른 입력: 금액 칩(+1만/+10만/+100만/+500만/+1천만) + 메모 칩(월세/카드값/용돈 등)

### 3-2. 자동이체
- 경로: 헤더 "이체" → "자동이체"
- 기능: 등록 목록 + 신규 등록 + 실행 이력 + 1회 예약이체 (자동이체 화면 안 버튼)
- 신규 등록: 출금계좌 → 입금계좌 verify → 주기(매월·매주·매일·1회) → 시작일·종료일·금액·메모

### 3-3. 자주 쓰는 계좌
- 경로: 헤더 "이체" → "자주 쓰는 계좌"
- 기능: 별명·메모 추가, 인라인 편집, 이체 시 prefill

## 4. 대출

### 4-1. 대출 상품 목록·신청
- 경로: 헤더 "대출" 또는 우측 사용자 메뉴 → 상품 카탈로그 → 대출 탭
- 단계: 상품 선택 → 사전심사(precheck) → 약관 동의 → 신청서 작성 → 첨부서류 4종 업로드 → 검토 대기

### 4-2. 대출 내 신청
- 경로: 헤더 "대출" → "신청 현황"
- 기능: 신청 단계별 상태(SUBMIT/APPROVED/REJECTED) + 첨부서류 보충 + 약정·실행 진행

### 4-3. 대출 계약 상세·상환
- 경로: 대출 → 계약 카드 클릭
- 기능: 원금·잔액·금리·다음 결제일·월 상환액 + 상환 스케줄 12회차 + 중도상환 신청

## 5. 보안

### 5-1. 보안 허브
- 경로: 우측 사용자 메뉴 → "보안 설정"
- 표시: 마지막 접속·OTP 등록 여부·등록 기기·의심 거래 4 슬롯

### 5-2. 이체·출금 한도 변경
- 경로: 보안 설정 → "이체·출금 한도"
- 단계: 계좌별 카드 → 한도 변경 신청 → OTP → 7일 점검 후 적용 (cooling-off)

### 5-3. OTP 등록
- 경로: 보안 설정 → "OTP 등록" 또는 `OTP 등록 화면`
- 단계: QR 코드 스캔(Google Authenticator) → 6자리 입력 → 검증

### 5-4. 간편 PIN 등록·재설정
- 경로: 보안 설정 → "간편 PIN"
- 단계: 6자리 키패드(셔플) → 확인 → 약한 PIN 거부

### 5-5. 의심 거래 (FDS 알림)
- 경로: 보안 설정 → "의심 거래"
- 기능: PENDING 거래 본인 확인 또는 신고 → status 전이

### 5-6. 기기 관리
- 경로: 보안 설정 → "등록 기기"
- 기능: 등록 기기 목록 + 새 기기 등록 (OTP 필요) + 기기 해지

## 6. 챗봇·고객 지원

### 6-1. 상담 챗봇
- 경로: 헤더 "챗봇"
- 기능: 약관·상품 FAQ 검색, LLM 답변, 상담원 연결 카드(LOW confidence 시)

### 6-2. 챗봇 히스토리
- 경로: 챗봇 화면 좌측 사이드 또는 "이전 대화"
- 기능: 날짜별 그룹(오늘/어제/지난 7일/30일/이전) + 검색

### 6-3. 챗봇 출처(약관) 보기
- 경로: 챗봇 답변 → 출처 카드 클릭
- 기능: 약관 전문 + 조항 검색 + 매칭 강조 + 공유 링크

## 7. 공지·이벤트·알림

### 7-1. 공지사항·이벤트
- 경로: 우측 사용자 메뉴 → "공지·이벤트"
- 기능: 검색 + 카테고리 칩(시스템/보안/서비스/정책 색 강조) + 진행상태(진행중/종료/예정) + 상세 페이지(공유 링크)

### 7-2. 알림 센터
- 경로: 우측 사용자 메뉴 → "알림 센터" 또는 헤더 종 아이콘
- 기능: 5탭(전체/거래/보안/대출/기타) + 미읽음 배지 + 일괄 읽음

## 8. 상품·가입

### 8-1. 상품 카탈로그
- 경로: 우측 사용자 메뉴 → "상품 카탈로그" (비로그인도 접근 가능)
- 기능: 4 타입 탭(SAVING/DEPOSIT/INSTALL/LOAN) + 검색 + 정렬(금리 높은 순 등) + FOREIGN 칩

### 8-2. 신규 가입 (회원가입)
- 경로: 로그인 화면 → "회원가입" 또는 `/signup`
- 단계: 약관 동의 → 본인인증 → 계정 정보(이메일·비번) → 가입 완료
- 별도: 외국인(`/signup/foreigner`) / 법인(`/signup/corporation`)

### 8-3. 상품 가입 (계좌 개설)
- 경로: 상품 카탈로그 → 상품 클릭 → "가입하기"
- 단계: 약관 5종 동의(전체 동의 1클릭) → 가입 정보 → 확인 → 완료
- 공동명의: 상품 상세 → "공동명의로 개설" / 미성년: "미성년 자녀 명의"

## 9. 비밀번호·계정 관리

### 9-1. 비밀번호 재설정
- 경로: 로그인 화면 → "비밀번호 재설정" 링크
- 단계: 이메일·이름·생년월일·통신사·휴대폰 5필드 → SMS 인증(mock 123456) → 새 비밀번호 → 완료

### 9-2. 로그아웃
- 경로: 우측 사용자 메뉴 → "로그아웃"
- 효과: JWT 즉시 블랙리스트, 캡쳐된 토큰도 401

## 10. 자주 듣는 질문 (직원 응대 시)

### Q: "이체 한도 변경은 어디서?"
→ "우측 상단 사용자 메뉴 → 보안 설정 → 이체·출금 한도 → 계좌 카드의 한도 변경 신청. 7일 점검 후 자동 적용됩니다."

### Q: "OTP 등록 어디서?"
→ "보안 설정 → OTP 등록 → Google Authenticator 앱으로 QR 스캔 후 6자리 코드 입력."

### Q: "공지사항 어디서 봐요?"
→ "헤더 우측 사용자 메뉴 → 공지·이벤트."

### Q: "대출 신청은?"
→ "헤더 대출 또는 상품 카탈로그 → 대출 탭 → 상품 선택 → 약관 → 신청서 → 첨부서류 업로드."

### Q: "통장 사본 출력?"
→ "계좌 목록 → 계좌 클릭 → 상단 통장 메뉴."

### Q: "FDS 의심 거래 본인 확인?"
→ "보안 설정 → 의심 거래 → 카드의 본인 거래 확인 또는 신고 버튼."

### Q: "자동이체 등록 어떻게?"
→ "헤더 이체 → 자동이체 → 신규 등록 버튼 → 출금계좌·입금계좌·주기·시작일·금액 입력."
