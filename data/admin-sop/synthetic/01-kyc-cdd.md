# KYC·CDD 본인확인 표준 절차 (직원 SOP)

> **AUDIENCE**: ADMIN  **CATEGORY**: KYC  **BASIS**: 금융실명거래 및 비밀보장에 관한 법률, 특정금융정보법 §5의2

## 1. 적용 대상
- 신규 계좌 개설 (예금·적금·외화·법인·공동명의·미성년 전 상품)
- 1억 원 이상 1회성 거래 (Enhanced Due Diligence)
- 의심거래 발생 후 재확인 요구 시
- 비대면 채널을 통한 거래 시 (가이드라인 §3.7 강화)

## 2. 표준 본인확인 단계 (Customer Due Diligence)

### 2-1. 1단계 — 신원확인서류 제출 요구
| 고객 유형 | 1차 서류 | 2차 서류 (선택) |
|---|---|---|
| 내국인 개인 | 주민등록증·운전면허증·여권 중 1종 | 건강보험증·재직증명서 |
| 외국인 | 외국인등록증·여권 | 재류카드·국내거소신고증 |
| 법인 | 사업자등록증·법인등기부등본 | 대표자 신분증 + 위임장 |
| 미성년 | 가족관계증명서 + 친권자 신분증 | 기본증명서 |

### 2-2. 2단계 — 진위 검증
1. 신분증 사진면 ↔ 본인 대조 (영상통화·창구 직접)
2. **마이데이터 행정안전부 진위확인 API** 호출 — 응답 코드별 분기
   - `0000` 정상 → 다음 단계
   - `0101` 만료 → 갱신 요구
   - `0202` 분실 신고 → 즉시 거래 중단, 본부 보고
3. 사진 일치도 90% 미만 시 추가 영상통화 또는 창구 방문 요구

### 2-3. 3단계 — 실제 소유자 확인 (법인·신탁만)
- 25% 이상 지분 보유자 또는 실질 지배자 신원 확인
- 정치적 주요인물(주요 고위공직자) 해당 여부 점검 → 해당 시 본부 컴플라이언스 사전 승인

### 2-4. 4단계 — 거래 목적·자금 출처 진술
- 거래 목적 항목 5종 중 선택 (생계/저축/사업/투자/기타)
- 1억 원 이상 입출금 발생 시 자금 출처 증빙 요구 (예: 매매계약서·근로소득원천징수영수증)

## 3. EDD (Enhanced Due Diligence) 적용 기준
다음 중 하나라도 해당 시 강화된 본인확인 적용:
- 외국인 고위험국가 (국제자금세탁방지기구 비협조 리스트) 거주자
- 주요 고위공직자 또는 그 가족
- 비대면 채널 1억 원 이상 신규 가입
- 가상자산사업자(가상자산사업자)와의 거래
- 자금 출처 진술과 거래 패턴 현저한 불일치

EDD 적용 시 본부 컴플라이언스부 사전 승인 + 6개월 거래 모니터링 강화.

## 4. 보존 기간
- KYC 기록·진위확인 응답·증빙 사본: **거래 종료 후 5년**
- EDD 기록: 거래 종료 후 **10년**
- 위반 시 특정금융정보법 §17 과태료 최대 1억 원

## 5. 위반 사례·주의 사항
- 신분증 발급일자 미확인 → 위조 신분증 적발 사례 다수
- 외국인 등록번호 체크섬 자동검증 누락 → 시스템 패치 후 필수
- 법인 대표자 외 임직원 명의로 개설하는 우회 가입 — STR 보고 대상

## 6. 발생 시 보고 라인
1. 1차 의심 → 지점 컴플라이언스 담당자
2. 본부 금융정보분석원 보고 대상 → 자금세탁방지센터 → 영업일 익일까지 보고서 작성
