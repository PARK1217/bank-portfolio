# 관리자 콘솔 화면 안내 (사이드바·메뉴 경로)

> **AUDIENCE**: ADMIN  **CATEGORY**: SCREEN_GUIDE  **목적**: 직원이 관리자 콘솔(5001) 에서 "어디서·어떻게" 질문 답변

## 1. 진입·인증

### 1-1. 관리자 로그인
- 경로: http://localhost:5001/login
- 사번 + 비밀번호 (예: ADMIN001 / admin1234)
- 성공 시 자동으로 대시보드(`/dashboard`) 진입

### 1-2. 사이드바 구조
- DA-ON BANK 로고 → 대시보드 (단일 항목)
- 그룹 4종:
  - 고객·계좌 (회원 검색·연체 추적·계좌 검색·거래내역·자동이체 모니터링)
  - 대출 (검토 큐·AI 의사결정·실행 계약·상환 내역)
  - 콘텐츠 (상품 관리·공지·이벤트·약관)
  - 리스크·운영 (의심거래 FDS·외부망 헬스·AI 관측 Phoenix·감사 로그)
- 단일 항목: AI 직원 어시스턴트 (사이드바 맨 아래)
- 좌측 하단: 본인 정보 + 로그아웃

## 2. 대시보드

### 2-1. 메인 KPI
- 경로: 사이드바 "대시보드"
- 표시: 검토 대기·오늘 자동 승인·연체 회원·외부망 정상·AI 관측 5장 카드 + 바로가기 카드 4개

## 3. 고객·계좌 관리

### 3-1. 회원 검색
- 경로: 사이드바 고객·계좌 → "회원 검색"
- 검색: 이름·고객번호·이메일·휴대폰 부분일치
- 상세: 클릭 시 회원 상세 화면 → 계좌 목록·거래 내역·민원 이력·등록 기기

### 3-2. 연체 추적
- 경로: 사이드바 고객·계좌 → "연체 추적"
- 기능: 연체 회원 집계(연체 건수·총 연체금액·최대 연체 일수) + 회원별 상세

### 3-3. 계좌 검색
- 경로: 사이드바 고객·계좌 → "계좌 검색"
- 검색: 계좌번호·예금주·상품 타입 필터

### 3-4. 거래내역 검색
- 경로: 사이드바 고객·계좌 → "거래내역 검색"
- 기능: 전체 거래 필터(계좌·기간·유형·금액 범위·상대 계좌)

### 3-5. 자동이체 모니터링
- 경로: 사이드바 고객·계좌 → "자동이체 모니터링"
- 기능: AUTO_TRANSFER 목록 + 실행 이력(SUCCESS/FAIL/DELAY) + 멱등 키 추적

## 4. 대출 관리

### 4-1. 검토 큐 (HUMAN_REVIEW)
- 경로: 사이드바 대출 → "검토 큐"
- 표시: AI 점수 0.40~0.85 회색지대 결정 (사람 검토 대기)
- 액션: 신청 상세 → ML 추론 결과 + 라벨링(APPROVE/REJECT + 메모)

### 4-2. AI 의사결정 이력
- 경로: 사이드바 대출 → "AI 의사결정"
- 기능: 자동 승인·반려·검토중 전체 결정 조회 + 필터(DECISION_CD)

### 4-3. 실행 계약
- 경로: 사이드바 대출 → "실행 계약"
- 기능: LOAN_CONTRACT 검색 + 잔액·금리·상환 스케줄·실행 이력

### 4-4. 상환 내역
- 경로: 사이드바 대출 → "상환 내역"
- 기능: 회차별 납입·연체·중도상환 이력

## 5. 콘텐츠 관리

### 5-1. 상품 관리
- 경로: 사이드바 콘텐츠 → "상품 관리"
- 기능: 상품 목록(필터 type/status) + KPI 3장 + 상품 등록(+ 새 상품 등록 버튼)
- 상세: 클릭 → 4 섹션(기본정보·기간 금리·금액 금리·우대조건·약관) + 상태 토글(SALE/SUSPEND/CLOSED)

### 5-2. 공지·이벤트
- 경로: 사이드바 콘텐츠 → "공지·이벤트"
- 기능: 공지·이벤트 목록 + 신규 작성 + 카테고리 색 강조

### 5-3. 약관 관리
- 경로: 사이드바 콘텐츠 → "약관 관리"
- 기능: 약관 마스터 + 버전 관리 + 상품 매핑

## 6. 리스크·운영

### 6-1. 의심거래 (FDS)
- 경로: 사이드바 리스크·운영 → "의심거래 (FDS)"
- 표시: FDS_DETECTION 큐 (PENDING/CONFIRM/REPORT/CLOSE) + 회원이름·점수·룰·LLM 설명
- 상세: 거래 컨텍스트 + AI_FDS_DECISION 합산 점수 + 조사 상태 변경

### 6-2. 외부망 헬스
- 경로: 사이드바 리스크·운영 → "외부망 헬스"
- 표시: 5종 외부 API(KFTC·BOK·MY_DATA·NICE·KCB) 최신 상태 + 시계열 차트

### 6-3. AI 관측 (Phoenix)
- 경로: 사이드바 리스크·운영 → "AI 관측 (Phoenix)"
- 임베드: Phoenix UI iframe — LLM 트레이스·Faithfulness 점수·토큰 사용량
- 새 탭: 상단 "새 탭으로 열기" 링크 → http://localhost:6006

### 6-4. 감사 로그
- 경로: 사이드바 리스크·운영 → "감사 로그"
- 기능: ADMIN_AUDIT_LOG 검색(직원·액션·날짜·결과) + 5년 보존

## 7. AI 직원 어시스턴트

### 7-1. AI 어시스턴트
- 경로: 사이드바 맨 아래 "AI 직원 어시스턴트"
- 기능: 내부 SOP·규정·법령 기반 RAG 검색 + LLM 단계별 답변
- 검색 범위: AUDIENCE_CD IN (USER, ADMIN, BOTH) — 직원은 모든 청크 접근

## 8. 자주 듣는 질문 (직원 본인 사용 시)

### Q: "사람이 검토할 대출 어디?"
→ "사이드바 대출 → 검토 큐. AI 점수 0.40~0.85 회색지대 결정만 노출."

### Q: "AI 자동 승인된 건 다 보고 싶어"
→ "사이드바 대출 → AI 의사결정 → 필터 'AUTO_APPROVE'."

### Q: "박철수 회원 검색"
→ "사이드바 고객·계좌 → 회원 검색 → '박철수' 입력 또는 회원번호 100001."

### Q: "FDS 의심 거래 처리"
→ "사이드바 리스크·운영 → 의심거래 (FDS) → 큐에서 클릭 → 조사 상태 CONFIRM/REPORT/CLOSE."

### Q: "감사 로그 누가 뭐 했는지 봐야 해"
→ "사이드바 리스크·운영 → 감사 로그 → 직원·액션·날짜·결과 필터."

### Q: "Phoenix LLM 추적 어디서?"
→ "사이드바 리스크·운영 → AI 관측 (Phoenix). iframe 안에 trace 검색·Faithfulness 점수 노출."

### Q: "상품 등록 어떻게?"
→ "사이드바 콘텐츠 → 상품 관리 → '+ 새 상품 등록' 버튼 → 4 카드 폼(기본정보·금액·기간·설명) → 자동 ID 발급."

### Q: "연체 회원 누구?"
→ "사이드바 고객·계좌 → 연체 추적. 최대 연체 일수 DESC 정렬, 클릭으로 회원별 상세."
