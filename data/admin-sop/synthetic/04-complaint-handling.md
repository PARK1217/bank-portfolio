# 민원 분류·이관·답변 가이드 (직원 SOP)

> **AUDIENCE**: ADMIN  **CATEGORY**: COMPLAINT  **BASIS**: 금융소비자보호법 §32, 금융분쟁조정세칙

## 1. 민원 분류 체계 (1차 접수 시)

### 1-1. 유형별
| 코드 | 유형 | 처리 부서 |
|---|---|---|
| C01 | 상품 약관·중요사항 설명 미흡 | 영업점 → 상품기획부 |
| C02 | 이자·수수료 부당 청구 | 영업점 → 정산부 |
| C03 | 직원 응대 불친절·언행 | 영업점 → 인사부 |
| C04 | 전산 오류·앱 장애 | 영업점 → 채널혁신부 |
| C05 | 보이스피싱 피해 신고 | 영업점 → 보안센터 |
| C06 | 개인정보 유출 의심 | 영업점 → CISO실 |
| C07 | 대출 심사 불복 | 영업점 → 여신심사부 |
| C08 | 분쟁조정 신청 | 영업점 → 분쟁조정담당 |

### 1-2. 긴급도별
- **즉시 (1시간 이내)**: C05 보이스피싱, C06 개인정보 유출
- **당일**: C04 전산 오류
- **3영업일**: C01~C03, C07
- **14영업일**: C08 분쟁조정

## 2. 표준 처리 절차

### 2-1. 1단계 — 접수
1. 「민원접수서」 작성 (필수 7항목)
2. **민원사실 인정 X / 부정 X — 사실 확인만**
3. 「민원 처리 안내문」 교부 — 처리 기한·담당자·연락처

### 2-2. 2단계 — 사실 조사
- 거래 내역·녹취·CCTV·시스템 로그 수집
- 관련 직원 진술 청취
- 외부 정보 필요 시 — 한국신용정보·통신사·결제대행사 조회

### 2-3. 3단계 — 답변
1. **표준 답변 기한**:
   - 일반: 14영업일 이내
   - 분쟁조정: 30일 이내
   - 보안·긴급: 즉시 1차 응답 + 5영업일 종결
2. 「민원 답변서」 작성 — 객관적 사실 + 처리 근거 약관·법령
3. 고객 만족도 조사 동봉

### 2-4. 4단계 — 이관 (지점에서 종결 불가 시)
- 14영업일 내 종결 불가 → 본부 민원관리부 이관
- 「이관통보서」 작성·접수번호 부여·고객에게 이관 사실 통지

## 3. 분쟁조정 신청 안내 (금감원·소비자원)

### 3-1. 신청 대상
- 자행 14일 답변에 불만족 시
- 본부 이관 후에도 미해결 시
- 1억 원 이하 금융 분쟁

### 3-2. 신청 방법
- 금감원 e-금융민원센터 (https://www.fcsc.kr)
- 한국소비자원 (https://www.kca.go.kr)
- 우편·방문도 가능

### 3-3. 처리 기한
- 금감원 분쟁조정위원회 — 60일 이내 결정
- 양측 수락 시 재판상 화해 효력

## 4. 답변문 작성 원칙

### 4-1. Do
- 객관적 사실만 기재
- 거래·로그 시간 정확히 명시 (예: "2026-05-26 14:23 출금 거래")
- 적용 약관·법령 명시 (예: "전자금융거래 이용약관 §15 ②")
- 정확한 사과 (불편 끼친 점) + 재발 방지 약속

### 4-2. Don't
- 책임 단정·반박
- 추측·감정 표현
- 다른 사례 비교
- 직원 개인 신상 노출

## 5. 금융소비자보호법 §32 핵심
1. 민원 접수일로부터 30일 이내 처리·통지 의무
2. 처리 지연 시 사유 안내 + 추가 처리 기간 약속
3. 동일·유사 민원 패턴 분기별 분석·재발방지 조치
4. 위반 시 영업정지·과징금 최대 5천만 원

## 6. 본부 보고 라인
- 일일 민원 현황 → 익일 09:00 본부 민원관리부 보고
- 주간 분석 — 유형·지점별 누적, 임원 보고
- 분기별 금감원 신고 — 1% 이상 처리지연·재발 케이스
