## 본문 작성 우선순위

본 코퍼스의 본문은 5종(1·3·5·6·7) 만 별도 파일로 작성. 나머지 4종(2·4·8·9)은 아래에 요약 윤곽만 — DB 세션이 시드 시 INSERT 하되 본문은 placeholder 또는 후속 단계에서 채움.

## DB 세션 작업 안내

1. `TERMS_MASTER` 9행 INSERT — 각 파일 상단의 메타(버전/시행일/카테고리) 사용.
2. `TERMS_CHANGE_HISTORY` — 6번 (전자금융) 만 v1→v2 이력 1행 추가.
3. `PRODUCT_TERMS_MAPPING` — 상품별 약관 매핑 (data/seed-products.md 참고):
   - SAVING 상품(P-101~105) → 5, 6, 7
   - DEPOSIT 상품(P-201~206) → 3, 6, 7
   - INSTALLMENT 상품(P-301~306) → 4, 6, 7
   - LOAN 상품(P-401~408) → 1, 2, 6, 7
4. 약관 본문 텍스트는 별도 `text` 컬럼이 없을 수 있음 — `TERMS_MASTER.TERMS_CONTENT` (또는 동등 컬럼)에 markdown 본문 그대로 저장. 챗봇 RAG 시 벡터화 대상.
5. 벡터 임베딩은 후속(인프라 RAG 세션 작업).

## 약관 2 — 가계대출 표준약관 (요약 윤곽)

여신거래기본약관(약관 1)을 가계대출에 특화. 핵심 추가 조항:
- 제1조 적용범위 — 가계 신용대출·전세대출·주택담보대출 등
- 제2조 대출 실행 — 본인 명의 입금계좌로만 입금
- 제3조 상환 방법 — 원리금균등 / 원금균등 / 만기일시상환 선택
- 제4조 중도상환 수수료 — 잔존원금 × 수수료율 (3년 이내 0.5%, 이후 면제)
- 제5조 자동상환 — 지정 출금계좌 잔액 부족 시 다음 영업일 재시도
- 제6조 신용평가 — 가계 신용대출은 매년 재평가, 조건 변경 가능

## 약관 4 — 적금 (정기적금 + 자유적립식) 약관 (요약 윤곽)

- 제1조 가입자격 — 만 14세 이상 (미성년은 친권자 동의)
- 제2조 납입 — 정기적금: 월 정액 자동이체 / 자유적립: 임의 입금
- 제3조 미납 — 정기적금 2회 연속 미납 시 우대금리 제외
- 제4조 중도해지 — 가입기간별 차등 (1년 미만 80% / 1년 이상 90% / 만기 100%)
- 제5조 만기 — 만기일 자동 지급 또는 자동 재가입(고객 선택)
- 제6조 우대금리 — 가입 시 등록한 우대조건 충족 여부에 따라 가산

## 약관 8 — 마케팅 정보 수신 동의 (요약 윤곽)

- 동의 항목: 신상품·금융혜택·이벤트·금리 변동 안내
- 수신 채널: SMS, 이메일, 앱 푸시, 우편 (선택 가능)
- 보관 기간: 동의일로부터 2년 (자동 만료, 갱신 요청 가능)
- 철회: 언제든 거래 메뉴에서 즉시 철회 가능

---

## 약관 9 — 자동이체 등록 동의 (요약 윤곽)

- 동의 사항: 출금 동의(출금계좌에서 자동 출금), 수취 동의(수취계좌로 자동 입금)
- 실행일이 영업일이 아닐 때: 다음 영업일 자동 이전 처리
- 잔액 부족: MAX_RETRY_COUNT(기본 3회)만큼 RETRY_INTERVAL_HOURS(24시간) 단위 재시도
- 해지: 자동이체 등록자가 언제든 즉시 해지 가능
