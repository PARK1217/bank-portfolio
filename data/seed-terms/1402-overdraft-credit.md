# 마이너스통장 한도여신 상품 특약약관

본 특약약관은 「마이너스통장 한도여신」(상품번호 402, 대출 전용 상품)에 적용되는 상품 특별 조항입니다. 본 상품은 상품 103(마이너스통장 한도형, 자유입출금예금과 결합)과 달리 **순수 한도여신**입니다.

## 제1조 (상품 개요)

| 항목 | 내용 |
|---|---|
| 상품명 | 마이너스통장 한도여신 |
| 상품 유형 | 한도여신 (revolving) |
| 가입 대상 | 만 19세 이상 직장인 / 사업자 |
| 약정 한도 | 1,000만원 ~ 1억원 |
| 약정 기간 | 1년 (자동 연장 가능) |
| 기본 금리 | 연 5.70% |
| 최저 우대 금리 | **연 5.10%** |
| 이자 부과 방식 | 일할 계산, 월말 자동 출금 |

## 제2조 (한도여신의 특징)

(1) **한도여신의 정의**

- 약정 한도 내에서 자유롭게 인출·상환 가능
- 인출한 금액에 대해서만 이자 부과
- 미사용 한도엔 이자 없음
- 약정 기간 내 무제한 재사용

(2) **상품 103과의 차이**

| 항목 | 상품 103 (마이너스통장 한도형) | 상품 402 (마이너스통장 한도여신) |
|---|---|---|
| 결합 | 자유입출금예금 + 대출 | 순수 대출 |
| 양수 잔액 | 가능 (예금 이자) | 불가 |
| 거래 자유도 | 일상 거래 통합 | 별도 대출 계좌 |
| 약정 한도 | 500만원~2억원 | 1,000만원~1억원 |

## 제3조 (가입 자격)

(1) **기본 자격**

- 만 19세 이상 ~ 만 65세 미만
- 재직 1년 이상 (또는 사업자 등록 1년 이상)
- 연소득 2,400만원 이상

(2) **신용 자격**

- 다온뱅크 내부신용등급 6등급 이상
- 채무불이행 정보 미등재
- 본인 명의 다온뱅크 자유입출금예금 계좌

## 제4조 (약정 한도)

| 내부 신용등급 | 약정 한도 범위 |
|---|---|
| 1~2등급 | 5,000만원 ~ 1억원 |
| 3등급 | 3,000만원 ~ 5,000만원 |
| 4~5등급 | 1,500만원 ~ 3,000만원 |
| 6등급 | 1,000만원 ~ 1,500만원 |

## 제5조 (금리 및 이자)

(1) **기본 금리**

연 5.70% (변동금리, COFIX 기준)

(2) **우대 금리**

| 우대 항목 | 우대 폭 |
|---|---|
| **급여이체 월 200만원 이상** | -0.30%p |
| **다온뱅크 적금 1,000만원 이상 보유** | -0.20%p |
| **자동상환 신청 (이자만)** | -0.10%p |

(3) **이자 산정**

```
일 이자 = 인출한 음수 잔액 × 약정 금리 / 365
월말에 누적 일 이자가 자동 출금
```

(4) **시뮬레이션 (한도 5,000만원 중 3,000만원 사용, 1개월 보유, 우대 금리 5.30%)**

```
일 이자 = 30,000,000 × 0.053 / 365 ≈ 4,356원
30일 누적 이자 = 약 130,685원
월말 자동 출금
```

## 제6조 (이자 출금 및 연체)

(1) **월말 이자 출금**

- 매월 말일 17:00 (한 달 누적 이자)
- 잔액 부족 시 다음 영업일 09:00 자동 재시도
- 영업일이 아닌 경우 다음 영업일

(2) **연체 처리**

| 경과기간 | 처리 |
|---|---|
| 1~5일 | SMS·앱 알림 |
| 6~30일 | 연체가산금리 3.0%p |
| 31~90일 | 연체가산금리 6.0%p, 신용정보 단기 등재 |
| 91일 이상 | 기한이익 상실, 잔액 전액 즉시 변제 |

## 제7조 (한도 갱신)

(1) **만기 갱신**

- 만기 1개월 전: SMS·앱 알림
- 만기 시 신용평가 재실시 (자동)
- 신용 변동에 따라 한도 조정 또는 갱신 거절

(2) **갱신 거절 시 처리**

- 갱신 거절 시 약정 종료 후 30일 이내 음수 잔액 전액 상환
- 30일 내 미상환 시 기한이익 상실

## 제8조 (중도상환 수수료)

본 상품은 한도여신이므로 **중도상환 수수료가 없습니다**. 자유롭게 일부·전부 상환 가능.

## 제9조 (해지 절차)

(1) 음수 잔액 전액 상환 → 약정 해지 가능

(2) **자동 정지 사유**

- 6개월 연속 미사용 (음수 잔액 0)
- 신용정보 등재
- 압류·가압류 발생

## 제10조 (특이사항)

(1) **본 상품의 활용 시나리오**

- 자영업자: 매출 변동성 큰 사업의 운영 자금
- 직장인: 갑작스러운 의료비·세금 등 단기 자금
- 부동산 중도금: 잔금 지급 전 임시 자금

(2) **추천 활용 팁**

- 한도만 받아두고 사용하지 않으면 이자 없음
- 사용 시 빠른 상환으로 이자 최소화
- 정기적 자금 필요 시 신용대출(상품 401) 권장

## 제11조 (예금자 보호)

본 상품은 「대출」이므로 「예금자보호법」 보호 대상이 아닙니다.

## 부칙

본 특약은 2026년 1월 1일부터 시행합니다.
