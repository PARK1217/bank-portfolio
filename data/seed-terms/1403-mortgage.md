# 주택담보대출 상품 특약약관

본 특약약관은 「주택담보대출」(상품번호 403)에 적용되는 상품 특별 조항입니다.

## 제1조 (상품 개요)

| 항목 | 내용 |
|---|---|
| 상품명 | 주택담보대출 |
| 상품 유형 | 담보부 가계대출 |
| 가입 대상 | 주택 보유자 또는 매매계약 체결자 |
| 대출 한도 | 1,000만원 ~ 10억원 (LTV·DSR 한도 내) |
| 대출 기간 | **10년 (120개월) / 20년 (240개월) / 30년 (360개월)** |
| 기본 금리 | 연 5.20% (10년 변동) ~ 5.60% (30년 고정) |
| 최저 우대 금리 | **연 4.40%** |
| 거치 기간 | 0 / 1 / 2 / 3년 (선택) |
| 상환 방식 | 원리금균등 / 원금균등 |

## 제2조 (가입 자격)

(1) **차주 자격**

- 만 19세 이상 ~ 만 70세 미만
- 대한민국 거주자
- 본인 명의 또는 부부 공동명의 주택 (배우자 동의)
- 다온뱅크 자유입출금예금 계좌 보유

(2) **담보 주택 자격**

- 아파트, 단독주택, 다세대주택, 빌라
- 등기 완료 또는 매매계약 체결
- 다른 금융기관 1순위 담보권 없음 (또는 1순위 변경 가능)

(3) **소득 자격**

- 연소득 2,400만원 이상
- 4대보험 가입 (또는 사업소득 증빙)
- 부부 합산 소득 인정

## 제3조 (대출 한도 (LTV·DSR))

(1) **LTV (Loan To Value, 담보인정비율)**

| 지역 | 1주택자 | 다주택자 |
|---|---|---|
| 비투기지역 | 70% | 60% |
| 조정대상지역 | 60% | 50% |
| 투기지역·투기과열지구 | 50% | 40% |

(2) **DSR (총부채원리금상환비율)**

```
DSR = (연간 원리금상환액 / 연간 소득) × 100
```

- 전 금융기관 가계대출 합산
- 한도: 40% (생계비 제외)

(3) **한도 산정 예시**

```
6억원 주택 (비투기지역, 1주택자):
- LTV 70%: 4억 2천만원
- 연소득 6,000만원 → DSR 40% = 연 2,400만원 한도
- 30년 원리금균등, 4.50%: 월 약 213만원 → 연 2,556만원
- DSR 한도 초과 → 한도 약 3억 9천만원으로 조정
```

## 제4조 (기간별 기본 금리)

| 대출 기간 | 변동금리 (연) | 고정금리 (연) | 최저 우대 (연) |
|---|---|---|---|
| 10년 (120개월) | 5.20% | 5.40% | 4.40% |
| 20년 (240개월) | 5.40% | 5.60% | 4.60% |
| 30년 (360개월) | 5.60% | 5.80% | 4.80% |

(1) **변동금리**: COFIX 6개월 변동

(2) **고정금리**: 약정 기간 전체 동일 금리 (5년 혼합형도 별도 운영)

## 제5조 (우대 금리)

| 우대 항목 | 우대 폭 |
|---|---|
| **신혼부부 (혼인 7년 이내)** | -0.40%p |
| **원리금 자동상환 신청** | -0.20%p |
| **다온뱅크 급여이체 + 1년 실적** | -0.20%p |

## 제6조 (상환 방식)

(1) **원리금균등** (기본)

매월 동일한 원리금 상환. 초기 이자 비중 큼, 후기 원금 비중 큼.

**예시 (3억원 × 30년 × 5.20%)**

```
월 상환액: 약 1,649,500원
총 상환액: 약 5억 9,382만원
총 이자: 약 2억 9,382만원
```

(2) **원금균등** (선택)

매월 동일한 원금 + 잔액 기준 이자. 총 이자 부담이 원리금균등 대비 적음.

**예시 (3억원 × 30년 × 5.20%)**

```
첫 월 상환액: 약 2,133,333원
마지막 월 상환액: 약 838,194원
총 상환액: 약 5억 3,468만원
총 이자: 약 2억 3,468만원
```

(3) **거치 후 분할 (선택)**

- 1년 거치: 12개월간 이자만 납입, 13개월부터 분할 상환
- 3년 거치: 36개월간 이자만 납입, 37개월부터 분할 상환
- 거치 기간 중 자유로운 부분 조기상환 가능

## 제7조 (담보 설정 비용)

| 항목 | 금액 |
|---|---|
| 등기 비용 | 약 50만원 (부동산 1건당) |
| 근저당권 설정비 | 약정금액의 0.2% (인지세 별도) |
| 감정평가 수수료 | 약 30만원 (1억 기준, 가치별 증감) |
| 보증료 (보증부 시) | 잔액의 0.3~0.7% (한국주택금융공사 보증) |

## 제8조 (실행 절차)

1. 모바일 앱 「대출 한도 조회」 (담보 주택 등기 정보 입력)
2. 본 심사 (재직·소득·신용평가 + 담보 감정)
3. 약정 (영업점 방문 필수)
4. 등기 절차 (법무사 협조)
5. 매도자 계좌로 잔금 입금

평균 처리 기간: **신청 ~ 실행까지 2~4주**

## 제9조 (중도상환 수수료)

| 약정 후 경과기간 | 수수료율 |
|---|---|
| 3년 이내 | **1.4%** (잔존원금 대비) |
| 3년 초과 | **면제** |

부분 조기상환은 매년 원금의 10% 이내 면제.

## 제10조 (연체 및 기한이익 상실)

| 연체기간 | 처리 |
|---|---|
| 1~5일 | SMS·앱 알림 |
| 6~30일 | 약정금리 + 3.0%p 연체이자 |
| 31~90일 | 약정금리 + 6.0%p 연체이자, 신용정보 등재 |
| 91일 이상 | 기한이익 상실, 담보권 실행 (경매) |

담보 가치가 30% 이상 하락 시 추가 담보 또는 일부 상환 요구.

## 제11조 (특이사항)

(1) **DSR 규제 강화**

「가계부채 관리 가이드라인」에 따라 신규 주담대는 DSR 40% 한도 적용. 1억원 초과 대출은 별도 심사.

(2) **갈아타기 (대환대출)**

다른 금융기관 주담대를 다온뱅크로 이전 가능 (대환). 별도 수수료 없음, 등기 비용만 발생.

## 제12조 (예금자 보호)

본 상품은 「대출」이므로 보호 대상 아님.

## 부칙

본 특약은 2026년 1월 1일부터 시행합니다.
