# 행복드림 정기예금 상품 특약약관

본 특약약관은 「행복드림 정기예금」(상품번호 201)에 적용되는 상품 특별 조항입니다. 본 특약에 정함이 없는 사항은 「정기예금 약관」(TERMS_ID 3)에 따릅니다.

## 제1조 (상품 개요)

| 항목 | 내용 |
|---|---|
| 상품명 | 행복드림 정기예금 |
| 상품 유형 | 거치식 정기예금 (단리) |
| 가입 대상 | 만 14세 이상 개인 |
| 가입 금액 | 100만원 ~ 100억원 |
| 가입 기간 | 6 / 12 / 24 / 36개월 (4종) |
| 기본 이율 (12개월 기준) | 연 3.55% |
| 최고 우대 이율 | 연 3.80% |
| 가입 채널 | 모바일 앱 / 인터넷뱅킹 / 영업점 |

## 제2조 (가입 자격)

- 만 14세 이상 (미성년자는 친권자 동의 후 가입)
- 본인 명의 다온뱅크 자유입출금예금 계좌 보유
- 1인 최대 가입 한도 없음 (다계좌 가입 가능)

## 제3조 (기간별 적용 이율)

| 가입 기간 | 기본 이율 (연) | 우대 충족 시 최고 (연) |
|---|---|---|
| 6개월 | 3.30% | 3.55% |
| 12개월 | 3.55% | 3.80% |
| 24개월 | 3.75% | 4.00% |
| 36개월 | 3.85% | 4.10% |

## 제4조 (우대 이율 조건)

(1) **우대 항목**

| 우대 항목 | 우대 폭 | 적용 시점 |
|---|---|---|
| **첫 거래 고객** (다온뱅크 첫 예적금 가입) | +0.10%p | 가입 시 즉시 |
| **만기 자동연장 신청** | +0.05%p | 가입 시 신청 |
| **비대면 가입** (모바일·인터넷) | +0.10%p | 가입 시 즉시 |

(2) **우대 적용 시 최고 이율**

```
기본 이율 (12개월)     : 연 3.55%
+ 첫 거래 우대         : +0.10%p
+ 만기 자동연장 우대   : +0.05%p
+ 비대면 우대          : +0.10%p
────────────────────
합계                  : 연 3.80%
```

(3) **우대 조건의 유지**

- 가입 시점에 충족된 우대는 만기까지 유지
- 만기 자동연장 신청을 가입 후 취소하는 경우 해당 우대 회수

## 제5조 (이자 산정 및 지급)

(1) **이자 산정 공식**

```
세전 이자 = 원금 × 적용 이율 × (예치 일수 / 365)
세금 (15.4%) = 세전 이자 × 0.154
세후 이자 = 세전 이자 - 세금
```

(2) **시뮬레이션 예시 (1,000만원 × 12개월 × 우대 3.80%)**

```
세전 이자 = 10,000,000 × 0.038 × 1 = 380,000원
세금       = 380,000 × 0.154 = 58,520원
세후 이자  = 321,480원
만기 수령액 = 10,000,000 + 321,480 = 10,321,480원
```

(3) **시뮬레이션 예시 (5,000만원 × 36개월 × 우대 4.10%)**

```
세전 이자 = 50,000,000 × 0.041 × 3 = 6,150,000원
세금       = 6,150,000 × 0.154 = 947,100원
세후 이자  = 5,202,900원
만기 수령액 = 55,202,900원
```

## 제6조 (이자 지급 방식)

가입 시 다음 중 선택:

| 방식 | 설명 |
|---|---|
| 만기일시지급 | 만기일에 원금+이자 일괄 지급 (기본) |
| 월이자지급 | 매월 이자 지정 계좌 지급, 만기에 원금만 |
| 분기이자지급 | 3개월마다 이자 지급 |
| 만기자동재예치 | 만기일에 동일 상품으로 자동 재가입 |

## 제7조 (중도해지)

(1) **중도해지 이율** (약정금리 대비, 기본 이율 기준)

| 경과 비율 | 적용 이율 |
|---|---|
| 1개월 미만 | 약정금리의 10% |
| 1~3개월 | 약정금리의 30% |
| 3~6개월 | 약정금리의 40% |
| 6~9개월 | 약정금리의 55% |
| 9~12개월 | 약정금리의 70% |
| 만기 80% 이상 | 약정금리의 90% |

(2) **우대 회수**

중도해지 시 가입 시 적용된 모든 우대금리는 전액 회수됩니다.

(3) **부득이한 사유의 특례**

다음 사유의 경우 약정금리의 80%까지 적용:

- 본인 사망·실종
- 천재지변
- 의료비·주거자금 등 입증되는 긴급생활자금

## 제8조 (만기 처리)

(1) **만기 자동 처리 옵션**

- **만기 자동지급**: 만기일에 원금+이자 지정 계좌로 입금
- **만기 자동연장**: 동일 기간으로 신규 가입 (가입 시점 기준 이율 적용)

(2) **만기 후 이율** (자동 처리 미신청 시)

| 만기 후 경과기간 | 적용 이율 |
|---|---|
| 1개월 이내 | 약정금리의 50% |
| 1~3개월 | 약정금리의 30% |
| 3개월 초과 | 연 0.10% (자유입출금 이율) |

(3) 만기 7일 전 SMS·앱 알림 발송

## 제9조 (예금자 보호)

「예금자보호법」 보호 대상. 1인당 5,000만원 한도 (다온뱅크 다른 예적금 합산).

## 제10조 (특이사항)

(1) **분할 해지 불가**: 본 상품은 가입 후 일부 인출이 불가능합니다. 자금이 필요한 경우 전액 해지하거나 「예금담보대출」(잔액의 최대 95%)을 이용할 수 있습니다.

(2) **추천 활용 시나리오**

- 단기 자금 운용: 6개월 ~ 12개월
- 결혼·자녀 학자금 등 목표 자금: 24개월 ~ 36개월
- 자금 분산 가입 (만기 분산): 6개월·12개월·24개월 각각 가입

## 부칙

본 특약은 2026년 1월 1일부터 시행합니다.
