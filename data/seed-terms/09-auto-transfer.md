# 자동이체 등록 동의

본 동의는 다온뱅크(이하 "은행")이 고객이 지정한 출금계좌에서 정기적으로 약정 금액을 자동 출금하여 지정 수취인에게 이체하는 권한을 부여하는 동의입니다.

본 동의는 자동이체를 등록·이용하는 모든 고객에게 공통 적용되며, 「전자금융거래법」 및 「자동이체등에관한법률」을 근거로 운영됩니다.

## 제1조 (자동이체의 정의 및 적용 범위)

(1) **자동이체의 정의**

자동이체는 고객이 사전 지정한 출금일·출금금액·수취인 정보에 따라 은행이 자동으로 자금을 이체하는 거래입니다.

(2) **적용 범위**

본 약관은 다음 모든 자동이체 거래에 공통 적용됩니다.

| 자동이체 종류 | 적용 거래 |
|---|---|
| **수신 자동납입** | 적금·정기예금 매월 납입 |
| **여신 자동상환** | 대출 원리금 상환, 카드 결제 |
| **공과금 자동납부** | 전기·가스·수도·통신·관리비·국세·지방세 |
| **카드 결제** | 신용카드·체크카드 대금 |
| **보험료·구독료** | 보험료, 통신비, 스트리밍 구독 |
| **CMS 출금** | 기업·기관의 정기 출금 (학원비, 회비 등) |

## 제2조 (출금 일정)

(1) **출금일 지정**

- 매월 출금일을 고객이 지정 (1일 ~ 28일 권장)
- 매월 말일 지정 시 → 해당 월의 마지막 영업일 (28~31일 변동)
- 영업일이 아닌 경우 (주말·공휴일) → 다음 영업일로 자동 조정
- 매주 / 매2주 / 분기별 등 주기 설정 가능

(2) **출금 시각**

- 평일: 09:00 ~ 17:00 사이 1회 출금 시도
- 1차 출금 실패 시 당일 14:00 / 17:00 추가 시도
- 주말·공휴일 출금일 지정 시 다음 영업일 09:00 출금

(3) **출금 알림**

- 출금 1일 전: SMS·앱 알림 (잔액 부족 예고 포함)
- 출금 당일: 출금 완료 또는 실패 알림
- 잔액 부족 시: 출금 시도 직후 SMS·앱 푸시

## 제3조 (잔액 부족 시 처리)

(1) **자동 재시도 정책**

| 재시도 | 시점 | 비고 |
|---|---|---|
| 1차 | 약정 출금일 09:00 | 정상 출금 시도 |
| 2차 | 잔액 부족 시 당일 14:00 | 잔액 보충 가능성 |
| 3차 | 잔액 부족 시 당일 17:00 | 마지막 시도 |
| 4차 | 다음 영업일 09:00 | 일부 거래에 한해 |

(2) **재시도 종료 후 처리**

모든 재시도 실패 시 미납 처리되며, 거래 종류별로 다음과 같은 영향이 발생합니다.

| 거래 | 미납 시 영향 |
|---|---|
| 적금 | 1회 미납: 안내 / 2회 연속: 우대금리 제외 |
| 대출 원리금 상환 | 약정금리에 연체가산금리 적용, 신용정보 단기 등재 |
| 카드 결제 | 다음 영업일 9시 자동 재시도, 3일 이상 연체 시 사용 정지 |
| 공과금 | 해당 기관의 미납 처리 절차 (가산금·정지 등) |
| 보험료 | 보험 효력 정지 (회사별 차이) |

(3) **즉시 재시도 (수동)**

잔액 보충 후 즉시 출금을 원하는 경우 모바일 앱에서 「수동 재시도」를 요청할 수 있습니다. 단, 출금일 당일에 한해 가능합니다.

## 제4조 (자동이체 등록·변경·해지)

(1) **등록 절차**

1. 모바일 앱 / 인터넷뱅킹 / 영업점 신청
2. 본인 인증 (생체인증 또는 OTP)
3. 출금계좌·수취인 정보·출금일·금액 입력
4. 자동이체 약정서 동의 (전자 서명)
5. 등록 완료 (즉시 발효)

(2) **변경 절차**

다음 항목은 즉시 변경 가능합니다.

- 출금일 (다음 회차부터 적용)
- 출금금액 (다음 회차부터 적용)
- 출금계좌 (다음 회차부터 적용)
- 일시 정지 (최대 3개월)

수취인 정보 변경은 기존 등록 해지 후 신규 등록이 필요합니다.

(3) **해지**

- 모바일 앱 / 인터넷뱅킹 / 영업점에서 즉시 해지 가능
- 해지 시 다음 회차부터 미실행
- 단, 해지 신청 당일 이미 처리된 출금은 일반 이체 절차로 환불 신청

(4) **출금일 당일 변경·해지의 효력**

출금일 당일 변경·해지 신청은 다음 회차부터 반영됩니다. 당일 출금은 정상 처리됩니다.

## 제5조 (수수료)

(1) **자동이체 수수료**

| 거래 유형 | 수수료 |
|---|---|
| 자동이체 등록·변경·해지 | 무료 |
| 당행 자동이체 (수취인이 다온뱅크) | 무료 |
| 타행 자동이체 (수취인이 타 은행) | 회당 100~200원 |
| CMS 출금 (기관 등록) | 기관 부담 |
| 외화 자동이체 | 1회당 5,000원 + 환율 우대 |

(2) **수수료 면제 우대**

다음 조건의 고객은 모든 자동이체 수수료가 면제됩니다.

- 급여이체 월 150만원 이상 등록
- 주거래 등급 SILVER 이상
- 정기예금·적금 5,000만원 이상 보유
- 신용대출 잔액 보유 (자동상환 자동이체 한정)
- 만 65세 이상 시니어 고객

## 제6조 (사고 책임)

(1) **본인 책임**

다음의 경우 손해는 본인이 부담합니다.

- 잘못된 수취인 정보 입력 (계좌번호·예금주명 오기)
- 잘못된 출금금액 등록
- 출금일 잘못 지정 (예: 급여일 전 출금)
- 본인이 등록한 자동이체로 인한 잔액 부족

(2) **은행 책임**

다음의 경우 손해는 은행이 부담합니다.

- 은행 시스템 장애로 출금 미실행
- 등록된 출금 정보를 임의로 변경
- 자동이체 약정서와 다른 금액 출금
- 보안 사고로 인한 부정 자동이체

(3) **착오 송금의 회수**

자동이체로 잘못 송금된 경우 다음 절차로 회수합니다.

1. 즉시 은행 콜센터 신고 (1599-XXXX)
2. 「착오송금 반환제도」 신청
3. 수취인 동의 시 즉시 반환
4. 수취인 거부 시 「예금보험공사 착오송금 반환지원」 신청

## 제7조 (개인정보 활용)

(1) 자동이체 등록 시 다음 정보가 수집·이용됩니다.

- 본인 식별정보 (성명, 본인확인값)
- 출금계좌 정보
- 수취인 계좌·예금주명·금융기관명
- 출금일·금액·주기

(2) 수취인 정보는 자동이체 약정 기간 동안만 보유되며, 약정 종료 후 1년 경과 시 익명화됩니다.

(3) 자동이체 거래 내역은 거래 종료일로부터 5년간 보유되며, 「국세기본법」에 따른 의무 기간입니다.

## 제8조 (분쟁 처리)

(1) 자동이체 관련 분쟁은 다음 순서로 처리합니다.

1. 영업점 또는 콜센터(1599-XXXX) 신청
2. 은행 민원처리부 정식 접수
3. 금융감독원 금융분쟁조정위원회 신청
4. 한국소비자원 분쟁조정
5. 법원 조정·소송

(2) 대표 분쟁 사례

| 분쟁 유형 | 해결 기준 |
|---|---|
| 미등록 자동이체 출금 | 「전자금융거래법」상 은행 책임, 환불 + 손해배상 |
| 출금일 지연 처리 | 시스템 점검 시점 확인, 차액 보상 |
| 수수료 면제 미적용 | 우대 조건 충족 증빙 후 환급 |
| 해지 후 출금 발생 | 즉시 환불 + 시정 조치 |

## 제9조 (특별 조항)

(1) **자동이체 약정 자동 갱신**

자동이체는 별도 종료 시점이 없는 한 본인이 해지할 때까지 무기한 자동 갱신됩니다.

(2) **본인 사망 시 처리**

본인 사망 시 자동이체는 다음과 같이 처리됩니다.

- 사망 신고 접수 즉시 모든 자동이체 일시 정지
- 상속인 청구 시 해지 처리
- 사망 후 6개월간 미청구 시 자동 해지

(3) **법령 변경 시**

「전자금융거래법」, 「자동이체등에관한법률」 등 관련 법령 변경 시 본 약관도 자동 적용됩니다.

## 부칙

본 동의는 2026년 1월 1일부터 시행합니다.
