# 단기 정기예금 상품 특약약관

본 특약약관은 「단기 정기예금」(상품번호 204)에 적용되는 상품 특별 조항입니다.

## 제1조 (상품 개요)

| 항목 | 내용 |
|---|---|
| 상품명 | 단기 정기예금 |
| 상품 유형 | 단기 거치식 정기예금 |
| 가입 대상 | 만 14세 이상 개인 / 법인 |
| 가입 금액 | 30만원 ~ 10억원 |
| 가입 기간 | 1 / 3 / 6개월 (3종) |
| 기본 이율 | 연 2.80% (1개월) / 연 3.00% (3개월) / 연 3.05% (6개월) |

## 제2조 (가입 자격)

- 만 14세 이상
- 본인 명의 다온뱅크 자유입출금예금 계좌
- 단기 자금 운용 목적 (전세 잔금·결혼 자금 등)

## 제3조 (기간별 이율)

| 가입 기간 | 기본 이율 (연) | 비고 |
|---|---|---|
| 1개월 | 2.80% | 가장 단기 |
| 3개월 | 3.00% | 분기 단위 운용 |
| 6개월 | 3.05% | 반기 단위 운용 |

본 상품은 우대 조건이 없는 단순 단기 운용 상품입니다.

## 제4조 (시뮬레이션)

(1) **5,000만원 × 1개월 × 2.80%**

```
세전 이자 = 50,000,000 × 0.028 × (30/365) ≈ 115,068원
세금       ≈ 17,720원
세후 이자  ≈ 97,348원
만기 수령액 ≈ 50,097,348원
```

(2) **1,000만원 × 3개월 × 3.00%**

```
세전 이자 = 10,000,000 × 0.030 × (90/365) ≈ 73,973원
세금       ≈ 11,392원
세후 이자  ≈ 62,581원
만기 수령액 ≈ 10,062,581원
```

(3) **3,000만원 × 6개월 × 3.05%**

```
세전 이자 = 30,000,000 × 0.0305 × (180/365) ≈ 451,233원
세금       ≈ 69,490원
세후 이자  ≈ 381,743원
만기 수령액 ≈ 30,381,743원
```

## 제5조 (이자 지급)

- **만기일시지급** (기본): 만기일에 원금+이자 일괄
- **만기재예치**: 만기일에 동일 상품으로 자동 재가입

## 제6조 (중도해지)

(1) **중도해지 이율** (단기 상품 특성상 차등 적용)

| 경과 비율 | 적용 이율 |
|---|---|
| 50% 미만 | 약정금리의 30% |
| 50~80% | 약정금리의 60% |
| 80% 이상 | 약정금리의 90% |

(2) 단기 상품은 만기 일수가 짧아 중도해지 시 손실이 더 크게 체감됩니다. 가급적 만기까지 보유를 권장합니다.

## 제7조 (만기 처리)

- 만기일에 자동 만기 처리 신청 시 즉시 원금+이자 지정 계좌 입금
- 만기 자동연장 신청 시 동일 상품으로 자동 재가입 (가입 시점 이율 적용)
- 만기 후 미해지 시 약정금리의 50% 적용 (1개월 이내)

## 제8조 (예금자 보호)

「예금자보호법」 보호 대상. 1인당 5,000만원 한도.

## 제9조 (추천 활용 시나리오)

(1) **1개월 단기 운용**

- 전세 잔금 지급 전 일시 보관
- 결혼 자금 단기 보관 (예식일 기준)
- 입금 직후 단기 운용

(2) **3개월 단기 운용**

- 분기별 자금 운용
- 학자금·등록금 대기 자금

(3) **6개월 단기 운용**

- 반기 자금 분리 운용
- 단기 채권 만기 대기

## 부칙

본 특약은 2026년 1월 1일부터 시행합니다.
