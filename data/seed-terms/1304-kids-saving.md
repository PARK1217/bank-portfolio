# 어린이 적금 상품 특약약관

본 특약약관은 「어린이 적금」(상품번호 304, **만 14세 미만 어린이 대상 자유적립식**)에 적용되는 상품 특별 조항입니다.

## 제1조 (상품 개요)

| 항목 | 내용 |
|---|---|
| 상품명 | 어린이 적금 |
| 상품 유형 | 자유적립식 (친권자 대리 가입) |
| 가입 대상 | **만 14세 미만 어린이** (친권자 대리) |
| 월 납입금액 | 1,000원 ~ 50만원 |
| 가입 기간 | 24 / 36 / 60개월 |
| 기본 이율 | 연 3.30% (24개월) ~ 3.70% (60개월) |
| 명의 | **어린이 본인 명의** (친권자가 대리) |

## 제2조 (가입 자격 및 절차)

(1) **가입 자격**

- 만 14세 미만 어린이 본인 명의
- 친권자(부 또는 모) 대리 가입

(2) **필수 서류**

- 어린이 기본증명서 (또는 가족관계증명서)
- 친권자 신분증
- 친권자 동의서 (인감 또는 본인서명)
- 친권자 명의 출금계좌 등록

(3) **공동 친권자 사인**

부모 모두 친권자인 경우 양친 동의가 원칙이나, 한 명의 친권자 동의만으로도 가입 가능 (단, 추후 분쟁 발생 시 양친 동의 입증 책임).

## 제3조 (기간별 이율)

| 가입 기간 | 기본 이율 (연) |
|---|---|
| 24개월 | 3.30% |
| 36개월 | 3.50% |
| 60개월 | 3.70% |

**우대 없는 단순 이율 상품** (어린이 친권자 대리이므로 우대 조건 평가 어려움).

## 제4조 (납입 방식)

(1) **자유적립식**: 시기·금액 자유 납입

(2) **납입 채널**: 모바일 앱·인터넷뱅킹·영업점·ATM

(3) **권장 활용**

- 출생 축하금 일시 거치
- 매월 용돈·세배돈 모음
- 어린이날·생일 등 기념일 단위 납입

## 제5조 (시뮬레이션)

**월 10만원 × 60개월 (총 600만원 납입, 3.70%)**

```
세전 이자 합계: 약 565,500원
세후 이자:    약 478,413원
만기 수령액:   약 6,478,413원 (어린이 12세 가입 → 17세 만기)
```

**월 30만원 × 36개월 (총 1,080만원 납입, 3.50%)**

```
세전 이자 합계: 약 599,400원
세후 이자:    약 507,084원
만기 수령액:   약 11,587,084원
```

## 제6조 (어린이 비과세종합저축)

(1) **만 18세 이하 비과세** (조건 충족 시)

「조세특례제한법」에 따라 어린이 비과세 종합저축 가입 가능.

- 가입 한도: 1인당 3,000만원
- 비과세 적용: 이자소득세 15.4% 면제
- 신청 절차: 가입 시 「비과세종합저축 가입신청서」 작성

(2) 비과세 신청 시 만기 시 세금 차감 없이 전액 수령.

## 제7조 (성년 시 명의 이전)

(1) 가입한 어린이가 만 14세 이상이 된 후, 본인이 직접 명의 관리할 수 있습니다.

(2) **명의 이전 절차**

- 만 14세 이상 도래 시 자동으로 본인 명의 관리 전환
- 만 19세 이상 도래 시 친권자 동의 없이 단독 거래 가능

(3) **만기 시 본인 수령**

가입 시 등록한 친권자 출금계좌가 아닌 본인 출금계좌로의 만기 수령 변경 가능 (만 19세 이상).

## 제8조 (중도해지)

| 경과 비율 | 적용 이율 |
|---|---|
| 1개월 미만 | 약정금리의 10% |
| 1~3개월 | 약정금리의 30% |
| 3~6개월 | 약정금리의 50% |
| 6개월~만기 80% | 약정금리의 70% |
| 만기 80% 이상 | 약정금리의 90% |

(1) 친권자 동의 시 중도해지 가능

(2) **어린이 명의 변경 (이혼·재혼 등)**

가족 관계 변동 시 친권자 동의 절차 변경. 법원 결정문 또는 가족관계증명서 갱신 필요.

## 제9조 (만기 처리)

(1) 만기일에 어린이 본인 또는 친권자 지정 계좌로 입금

(2) 만 14세 미만이 만기인 경우 친권자가 수령 처리

(3) 만 14세 이상이 만기인 경우 본인이 수령 (친권자 동의 불필요)

## 제10조 (예금자 보호)

「예금자보호법」 보호 대상. 1인당 5,000만원 한도 (어린이 명의 기준).

## 부칙

본 특약은 2026년 1월 1일부터 시행합니다.
