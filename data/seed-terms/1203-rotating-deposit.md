# 회전식 정기예금 상품 특약약관

본 특약약관은 「회전식 정기예금」(상품번호 203)에 적용되는 상품 특별 조항입니다. 본 상품은 12개월마다 자동으로 시장 금리를 반영해 재예치되는 회전형 정기예금입니다.

## 제1조 (상품 개요)

| 항목 | 내용 |
|---|---|
| 상품명 | 회전식 정기예금 |
| 상품 유형 | 회전형 정기예금 (12개월 회전) |
| 가입 대상 | 만 14세 이상 개인 |
| 가입 금액 | 100만원 ~ 10억원 |
| 회전 주기 | 12개월 |
| 최장 운용 기간 | 60개월 (5회 회전) |
| 기본 이율 | 연 3.30% (2026-01-01 시점) |
| 가입 채널 | 모바일 앱 / 인터넷뱅킹 / 영업점 |

## 제2조 (회전형 운용의 정의)

(1) **회전형 운용**은 12개월 단위로 자동 재예치되며, 매 회전 시점의 시장 금리를 반영하여 새로운 이율이 적용되는 운용 방식입니다.

(2) **장점**

- 시장 금리 상승 시 자동 반영
- 별도 만기 갱신 절차 불필요
- 12개월마다 일부·전부 해지 가능 (회전 시점 = 무위약 해지 가능 시점)

(3) **단점**

- 시장 금리 하락 시 자동 인하
- 회전 시점 외 해지 시 중도해지 이율 적용

## 제3조 (이율 산정)

(1) **각 회전 차수별 적용 이율**

매 회전 시점(가입 후 12개월, 24개월, 36개월, 48개월)의 시장 기준금리에 가산금리를 더해 결정됩니다.

```
회전 시점 적용 이율 = 시장 기준금리 (한국은행 기준) + 가산금리 (연 0.30%p)
```

(2) **이율 통지**

- 회전 시점 1개월 전: SMS·앱 알림 (예상 이율 안내)
- 회전 시점 당일: 확정 이율 통지
- 회전 후 첫 월: 통장에 적용 이율 표기

(3) **시뮬레이션 예시 (예측)**

```
1차 (2026-01) 가입 이율  : 연 3.30%
2차 (2027-01) 회전 이율  : 연 3.50% (예측, 시장 금리 상승)
3차 (2028-01) 회전 이율  : 연 3.20% (예측, 금리 하락)
4차 (2029-01) 회전 이율  : 연 3.40%
5차 (2030-01) 회전 이율  : 연 3.50%
```

## 제4조 (이자 산정 및 지급)

(1) **이자 산정 (각 회전 기간별)**

```
세전 이자 = 원금 × 적용 이율 × (예치 일수 / 365)
```

(2) **지급 방식**

- **각 회전 시점에 분기 이자 지급** (기본)
- **누적 재예치**: 회전 시점에 이자가 원금에 합산되어 다음 12개월 운용
- **만기일시지급**: 5회 회전 종료 후 총 지급

(3) **누적 재예치 효과 (1억원 × 5회전)**

```
1차 (3.30%): 원금 1억원 → 만기 1억 330만원
2차 (3.50%): 원금 1억 330만원 → 만기 1억 692만원
3차 (3.20%): 원금 1억 692만원 → 만기 1억 1,034만원
4차 (3.40%): 원금 1억 1,034만원 → 만기 1억 1,409만원
5차 (3.50%): 원금 1억 1,409만원 → 만기 1억 1,808만원

5년 누적 세전 이자 : 약 1,808만원
세후 (15.4% 적용)  : 약 1,530만원
```

## 제5조 (해지 가능 시점)

(1) **회전 시점 해지 (무위약)**

매 회전 시점 (가입일 12개월/24개월/36개월/48개월)에는 약정 이율 100% 적용 + 우대 유지 + 위약 없음.

(2) **회전 시점 외 해지 (중도해지)**

| 경과 비율 (해당 회전 기간 내) | 적용 이율 |
|---|---|
| 1개월 미만 | 약정금리의 10% |
| 1~3개월 | 약정금리의 30% |
| 3~6개월 | 약정금리의 40% |
| 6~9개월 | 약정금리의 55% |
| 9~12개월 | 약정금리의 70% |

회전 시점에서만 무위약 해지 가능하다는 점이 본 상품의 핵심입니다.

## 제6조 (자동 만료)

(1) **5회 회전 완료 후**

본 상품은 5회 회전 (총 60개월) 완료 시 자동 만료되며, 다음과 같이 처리됩니다.

- 만기 자동 해지: 원금+이자 지정 계좌로 입금
- 만기 자동 재가입: 본 상품 동일 조건으로 재가입 (5회 추가)

(2) **만기 자동 처리 옵션 변경**

가입 후 만기 자동 처리 옵션은 영업점·앱에서 1년에 1회 변경 가능합니다.

## 제7조 (장점·단점 안내)

(1) **장점**

- 시장 금리 상승 시 자동 반영
- 매 12개월마다 무위약 해지 가능
- 만기 갱신 절차 불필요 (자동 진행)
- 누적 재예치로 복리 효과 (옵션 선택 시)

(2) **단점**

- 시장 금리 하락 시 자동 인하
- 일반 정기예금 (상품 201) 대비 가산금리 낮음
- 회전 시점 외 해지 시 중도해지 적용

(3) **추천 시나리오**

- 5년 이상 장기 운용 계획
- 금리 변동 예측이 어려운 환경
- 12개월마다 자금 일부 활용 가능성 있는 경우

## 제8조 (예금자 보호)

「예금자보호법」 보호 대상. 1인당 5,000만원 한도.

## 부칙

본 특약은 2026년 1월 1일부터 시행합니다.
