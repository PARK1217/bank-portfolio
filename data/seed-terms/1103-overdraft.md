# 마이너스통장 한도형 상품 특약약관

본 특약약관은 「마이너스통장 한도형」(상품번호 103)에 적용되는 상품 특별 조항입니다. 본 특약은 자유입출금예금과 한도여신이 결합된 복합 상품으로, 「자유입출금예금 약관」(TERMS_ID 5), 「여신거래기본약관」(TERMS_ID 1), 「가계대출 표준약관」(TERMS_ID 2)에 함께 따릅니다.

## 제1조 (상품 개요)

| 항목 | 내용 |
|---|---|
| 상품명 | 마이너스통장 한도형 |
| 상품 유형 | 자유입출금 + 한도여신 복합 |
| 가입 대상 | 만 19세 이상, 재직 1년 이상, 신용등급 6등급 이상 |
| 약정 한도 | 500만원 ~ 2억원 (신용등급별 차등) |
| 약정 기간 | 1년 (자동 연장 가능) |
| 기본 대출 금리 | 연 6.50% (신용등급 4~5등급 기준) |
| 우대 적용 시 | 연 5.50% ~ 6.20% |
| 예금 잔액 이율 | 연 0.10% (음수 잔액이 아닐 때) |

## 제2조 (한도여신의 정의)

(1) 본 상품은 자유입출금 잔액이 0 이하로 떨어질 경우, 약정 한도까지 자동으로 대출이 실행되는 「한도여신」 방식입니다.

(2) 음수 잔액에 대해서는 약정 대출 금리가 일할 계산으로 부과됩니다.

(3) 양수 잔액 회복 시 자동으로 음수 잔액이 상환되며, 잔액 변동 시점부터 이자가 정지됩니다.

## 제3조 (가입 자격)

(1) **기본 자격**

- 만 19세 이상
- 대한민국 거주자 (외국인은 별도 심사)
- 재직 1년 이상 (재직증명서 + 4대보험 증빙)
- 연소득 2,400만원 이상
- 본인 명의 다온뱅크 자유입출금예금 계좌 보유

(2) **신용 자격**

- 다온뱅크 내부신용등급 6등급 이상
- 「신용정보의 이용 및 보호에 관한 법률」상 채무불이행 정보 미등재
- 다른 금융기관의 채무불이행 미등재

## 제4조 (약정 한도)

(1) **신용등급별 한도 범위**

| 내부 신용등급 | 약정 한도 |
|---|---|
| 1~2등급 | 5,000만원 ~ 2억원 |
| 3등급 | 3,000만원 ~ 1억원 |
| 4등급 | 1,500만원 ~ 5,000만원 |
| 5등급 | 800만원 ~ 2,000만원 |
| 6등급 | 500만원 ~ 1,000만원 |

(2) **한도 산정 요소**

- 연소득의 50% ~ 200% 범위
- 기존 부채 및 DSR (40% 한도)
- 다온뱅크 거래 실적
- 재직 기간 및 직장 안정성

## 제5조 (대출 금리)

(1) **금리 구성**

```
적용금리 = 기준금리 (COFIX) + 가산금리 - 우대금리
```

(2) **기본 금리 (등급별)**

| 신용등급 | 기본 금리 (가산금리 포함) |
|---|---|
| 1~2등급 | 연 5.50% ~ 6.00% |
| 3등급 | 연 6.00% ~ 6.50% |
| 4~5등급 | 연 6.50% ~ 7.00% |
| 6등급 | 연 7.00% ~ 7.50% |

(3) **우대 금리**

| 우대 항목 | 우대 폭 |
|---|---|
| 급여이체 월 200만원 이상 (한도 +500만원 동시 적용) | -0.30%p |
| 다온뱅크 적금 5,000만원 이상 보유 | -0.20%p |
| 다온뱅크 신용카드 월 30만원 이상 사용 | -0.20%p |
| 마이데이터 연동 | -0.10%p |

(4) **이자 산정**

```
일 이자 = 음수 잔액 × 약정 금리 / 365
월말에 누적 일 이자가 자동 출금됨
```

## 제6조 (이자 출금 및 연체)

(1) **이자 출금 시점**

- 매월 말일 17:00 (한 달 누적 이자)
- 잔액 부족 시 다음 영업일 09:00 자동 재시도
- 영업일이 아닌 경우 다음 영업일

(2) **연체 처리**

| 경과기간 | 처리 |
|---|---|
| 1~5일 | SMS·앱 알림 |
| 6~30일 | 약정금리 + 연체가산금리 3%p, 신용정보 단기 등재 |
| 31~90일 | 연체가산금리 6%p, 신용정보 등재 |
| 91일 이상 | 기한이익 상실, 잔액 전액 즉시 변제 청구 |

(3) **기한이익 상실 시 처리**

- 음수 잔액 전액 즉시 변제기 도달
- 본인 다른 예적금 (정기예금·적금) 자동 상계 가능 (사전 동의 시)
- 강제집행 절차 개시

## 제7조 (한도 갱신 및 변경)

(1) **만기 갱신**

- 만기 1개월 전부터 SMS·앱 알림 발송
- 만기 시 신용평가 재실시 (자동)
- 신용 변동 시 한도 조정 가능 (축소·동결·증액)
- 갱신 미신청 시 만기일에 자동 해지

(2) **한도 변경 신청**

- 신용평가 결과 1등급 이상 상승 시 한도 증액 신청 가능
- 신용평가 결과 2등급 이상 하락 시 한도 축소 통지

## 제8조 (해지 및 자동 정지)

(1) **본인 해지**

- 음수 잔액 전액 상환 후 즉시 해지 가능
- 해지 시 한도 자동 소멸

(2) **자동 정지 사유**

- 6개월 연속 한도 미사용 (음수 잔액 0)
- 신용정보 등재
- 다른 금융기관의 압류·가압류

## 제9조 (중도상환 수수료)

본 상품은 한도여신 방식으로 **중도상환 수수료가 없습니다**. 음수 잔액을 언제든 자유롭게 상환할 수 있습니다.

## 제10조 (예금자 보호)

본 상품의 「자유입출금예금」 부분(양수 잔액)은 「예금자보호법」에 따라 예금보험공사가 보호합니다. 「대출」 부분 (음수 잔액)은 보호 대상이 아닙니다.

## 부칙

본 특약은 2026년 1월 1일부터 시행합니다.
