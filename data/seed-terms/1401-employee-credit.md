# 직장인 우대 신용대출 상품 특약약관

본 특약약관은 「직장인 우대 신용대출」(상품번호 401)에 적용되는 상품 특별 조항입니다. 본 특약에 정함이 없는 사항은 「가계대출 표준약관」(TERMS_ID 2) 및 「여신거래기본약관」(TERMS_ID 1)에 따릅니다.

## 제1조 (상품 개요)

| 항목 | 내용 |
|---|---|
| 상품명 | 직장인 우대 신용대출 |
| 상품 유형 | 무보증 신용대출 |
| 가입 대상 | 만 19세 이상 직장인 |
| 대출 한도 | 500만원 ~ 1억원 (신용등급별 차등) |
| 대출 기간 | 12 / 24 / 36 / 48 / 60개월 |
| 기본 금리 | 연 4.50% (12개월) ~ 5.20% (60개월) |
| 최저 우대 금리 | **연 3.50%** (모든 우대 충족 시) |
| 상환 방식 | 원리금균등 / 만기일시 |

## 제2조 (가입 자격)

(1) **기본 자격**

- 만 19세 이상 ~ 만 65세 미만
- 대한민국 거주자 (외국인은 상품 408 이용)
- **재직 6개월 이상** (재직증명서 + 4대보험 가입증명서)
- 연소득 2,400만원 이상

(2) **신용 자격**

- 다온뱅크 내부신용등급 7등급 이상
- 「신용정보의 이용 및 보호에 관한 법률」상 채무불이행 정보 미등재
- 다른 금융기관 연체 정보 없음
- 본인 명의 다온뱅크 자유입출금예금 계좌

## 제3조 (대출 한도)

(1) **신용등급별 한도**

| 내부 신용등급 | 한도 범위 | 연소득 대비 |
|---|---|---|
| 1~2등급 | 5,000만원 ~ 1억원 | 200% |
| 3등급 | 3,000만원 ~ 5,000만원 | 150% |
| 4등급 | 1,500만원 ~ 3,000만원 | 100% |
| 5등급 | 800만원 ~ 1,500만원 | 70% |
| 6등급 | 500만원 ~ 800만원 | 50% |
| 7등급 | 500만원 (한도) | 30% |

(2) **DSR 한도**: 40% (총부채원리금상환비율, 본인+가족 합산)

## 제4조 (기간별 기본 금리)

| 대출 기간 | 기본 금리 (연) | 최저 우대 (연) |
|---|---|---|
| 12개월 | 4.50% | 3.50% |
| 24개월 | 4.70% | 3.70% |
| 36개월 | 4.85% | 3.85% |
| 48개월 | 5.00% | 4.00% |
| 60개월 | 5.20% | 4.20% |

## 제5조 (우대 금리 조건)

(1) **우대 항목 (대출이므로 음수 = 인하)**

| 우대 항목 | 우대 폭 |
|---|---|
| **급여이체 월 250만원 이상 + 6개월 실적** | -0.30%p |
| **원리금 자동상환 신청** | -0.20%p |
| **내부 신용등급 1~3등급** | -0.50%p |

(2) **최저 우대 금리 산정 예시 (24개월 기준)**

```
기본 금리 (24개월)  : 연 4.70%
- 급여이체 우대     : -0.30%p
- 자동상환 우대     : -0.20%p
- 1~3등급 우대      : -0.50%p
────────────────────
적용 금리           : 연 3.70%
```

## 제6조 (상환 방식)

(1) **원리금균등** (기본)

매월 동일한 원리금 상환. 초반엔 이자 비중이 크고 후반엔 원금 비중이 큼.

**예시 (3,000만원 × 36개월 × 4.85% 우대)**

```
월 상환액: 약 897,283원
총 상환액: 32,302,188원
총 이자: 2,302,188원
```

(2) **만기일시** (선택)

매월 이자만 납입, 만기일에 원금 일시 상환. 단기 자금 운용에 적합.

**예시 (3,000만원 × 12개월 × 4.50% 우대)**

```
월 이자: 약 112,500원
12개월 누적 이자: 약 1,350,000원
만기 원금 일시 상환: 30,000,000원
```

## 제7조 (실행 절차)

(1) **신청 단계**

1. 모바일 앱 「대출 한도 조회」 (가심사 = 신용조회 없음)
2. 본 심사 신청 (재직·소득 증빙 업로드)
3. 신용평가 (다온뱅크 자체 + 신용평가기관)
4. 약정 (전자약정서 동의)
5. 본인 명의 다온뱅크 자유입출금예금에 입금

(2) **실행 시간**

- 평일 09:00 ~ 17:00 실행
- 가심사 ~ 실행까지 평균 1~3 영업일

## 제8조 (중도상환 수수료)

| 약정 후 경과기간 | 수수료율 |
|---|---|
| 1년 이내 | 0.5% (잔존원금 대비) |
| 1~2년 | 0.4% |
| 2~3년 | 0.3% |
| 3년 초과 | **면제** |

부분 조기상환은 매년 원금의 10% 이내 면제.

## 제9조 (연체 처리)

| 연체기간 | 처리 |
|---|---|
| 1~5일 | SMS·앱 알림 |
| 6~30일 | 약정금리 + 3.0%p 연체이자 |
| 31~90일 | 약정금리 + 6.0%p 연체이자, 신용정보 단기 등재 |
| 91일 이상 | 기한이익 상실, 채권 추심 절차 |

## 제10조 (조건 변경)

가입 후 1년 경과 시 다음 조건 변경 신청 가능:

- 금리 유형 변경 (변동 ↔ 고정, 변동수수료 0.7%)
- 상환 방식 변경 (원리금균등 ↔ 만기일시, 무료)
- 상환일 변경 (월 1회 무료)
- 한도 증액 (신용평가 재실시)

## 제11조 (특이사항)

(1) **본 상품의 직장인 우대 특징**

- 일반 신용대출 (시중 평균 6~8%) 대비 1~2%p 낮은 금리
- 우대 충족 시 정책성 서민금융 (사잇돌·새희망홀씨)보다 낮음
- 신용등급 1~3등급 직장인이 가장 유리

(2) **재직 증빙 우대**

- 4대보험 가입 + 재직 3년 이상: 추가 -0.10%p 검토
- 공무원·교사·공기업: 별도 우대 (영업점 문의)

## 제12조 (예금자 보호 안내)

본 상품은 「대출」이므로 「예금자보호법」 보호 대상이 아닙니다 (예금자 보호는 예적금에만 적용).

## 부칙

본 특약은 2026년 1월 1일부터 시행합니다.
