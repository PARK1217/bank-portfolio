# 자유적립 적금 상품 특약약관

본 특약약관은 「자유적립 적금」(상품번호 302)에 적용되는 상품 특별 조항입니다. 본 상품은 가입 기간 내 자유롭게 납입할 수 있는 자유적립식 적금입니다.

## 제1조 (상품 개요)

| 항목 | 내용 |
|---|---|
| 상품명 | 자유적립 적금 |
| 상품 유형 | 자유적립식 (시기·금액 자유 납입) |
| 가입 대상 | 만 14세 이상 개인 / 법인 |
| 1회 최소 납입 | 1,000원 |
| 월 최대 납입 | 100만원 |
| 가입 기간 | 12 / 24 / 36개월 |
| 기본 이율 | 연 3.35% (12개월) ~ 3.70% (36개월) |
| 최고 우대 이율 | 연 3.80% (비대면 가입 시) |

## 제2조 (자유적립의 특징)

(1) **정기적금과의 차이**

| 항목 | 정기적금 (301) | 자유적립 (302) |
|---|---|---|
| 납입 방식 | 매월 정액 자동이체 | 자유롭게 (시기·금액) |
| 약정 금액 | 가입 시 고정 | 1회 최소 1,000원만 |
| 미납 개념 | 있음 | 없음 |
| 우대 평가 | 미납 횟수 | 누적 납입 실적 |

(2) **자유적립의 장점**

- 자금 여유에 따라 자유롭게 납입
- 보너스·상여금 입금 시 일시 거치
- 미납 페널티 없음

## 제3조 (납입 채널 및 한도)

(1) **납입 채널**

- 모바일 앱 / 인터넷뱅킹 (즉시 이체)
- 영업점 창구
- ATM (현금 입금)
- 자동이체 (선택, 정기 납입 원할 경우)

(2) **납입 한도**

- 1회 최소 1,000원
- 1회 최대 100만원
- 월 최대 100만원
- 가입 기간 총 한도 = 월 100만원 × 가입 개월수

## 제4조 (기간별 이율 및 우대)

| 가입 기간 | 기본 이율 (연) | 우대 충족 시 최고 |
|---|---|---|
| 12개월 | 3.35% | 3.45% |
| 24개월 | 3.55% | 3.65% |
| 36개월 | 3.70% | 3.80% |

**우대 조건**: 비대면 가입 시 +0.10%p (가입 시 즉시 적용)

## 제5조 (이자 산정)

(1) **각 회차별 이자 산정**

```
회차별 이자 = 회차별 납입금액 × 적용 이율 × (회차 납입일부터 만기일까지 일수 / 365)
총 이자 = ∑ 회차별 이자
```

(2) **시뮬레이션 (월 평균 30만원 × 12개월 = 360만원, 우대 3.45%)**

```
가입일에 30만원 + 매월 1일 30만원 납입 (총 12회):
1회차 (12개월): 300,000 × 0.0345 × 365/365 = 10,350원
2회차 (11개월): 300,000 × 0.0345 × 334/365 ≈ 9,470원
...
12회차 (1개월): 300,000 × 0.0345 × 31/365 ≈ 879원

세전 이자 합계: 약 67,275원
세후 이자:    약 56,917원
만기 수령액:   약 3,656,917원
```

## 제6조 (우대 평가 (자유적립식 특칙))

(1) **우대 평가 기준**

자유적립식의 우대는 다음 조건을 충족해야 만기 시 적용됩니다.

- 가입 기간 50% 이상에 걸쳐 납입 실적 (월 1회 이상)
- 가입 기간 총 납입금액이 「월 100만원 × 가입 개월수 × 50%」 이상

(2) **예시 (12개월 가입 기준)**

```
조건 충족: 12개월 중 6개월 이상 납입, 누적 600만원 이상
조건 미달: 가입 기간 마지막 1개월에만 일시 납입 → 우대 제외
```

## 제7조 (중도해지)

| 경과 비율 | 적용 이율 |
|---|---|
| 1개월 미만 | 약정금리의 10% |
| 1~3개월 | 약정금리의 30% |
| 3~6개월 | 약정금리의 50% |
| 6개월~만기 80% | 약정금리의 70% |
| 만기 80% 이상 | 약정금리의 90% |

## 제8조 (만기 처리)

- 만기일 원금 + 이자 지정 계좌 입금 (기본)
- 만기 자동 재가입 시 동일 상품으로 재가입
- 만기 1개월 후 자유입출금예금 자동 전환

## 제9조 (예금자 보호)

「예금자보호법」 보호 대상. 1인당 5,000만원 한도.

## 제10조 (추천 활용 시나리오)

- 매월 일정 금액이 정해지지 않는 변동 소득자 (프리랜서·자영업자)
- 보너스·상여금 일시 적립
- 일시 자금 단기 운용 (월 한도 내)
- 자녀 용돈·연말정산 환급금 자동 적립

## 부칙

본 특약은 2026년 1월 1일부터 시행합니다.
