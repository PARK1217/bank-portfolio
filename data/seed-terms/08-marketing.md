# 마케팅 정보 수신 동의

본 동의서는 「개인정보 보호법」 제22조 및 「정보통신망 이용촉진 및 정보보호 등에 관한 법률(이하 정보통신망법)」 제50조에 따라, 다온뱅크(이하 "은행")이 고객에게 마케팅 목적의 정보를 발송하기 위해 필요한 동의를 받기 위한 것입니다.

본 동의는 **선택 사항**이며, 동의하지 않더라도 다온뱅크의 모든 금융 서비스 이용에 영향이 없습니다.

## 제1조 (목적 및 적용 범위)

(1) 본 동의서는 다음 마케팅 활동에 적용됩니다.

- 신상품·금리·이벤트 안내
- 우대 혜택 및 할인·캠페인 정보
- 맞춤형 금융 상품 추천
- 기념일·생일 등 개인화 메시지
- 제휴사 금융 상품 안내 (별도 동의 필요)

(2) 본 동의서는 「전자금융거래법」상 거래 안내(이체 알림, 보안 알림)에는 적용되지 않습니다. 거래 안내는 본 동의와 별개로 모든 고객에게 발송됩니다.

## 제2조 (수신 동의 항목)

(1) **마케팅 정보의 종류**

| 분류 | 세부 내용 |
|---|---|
| **신상품 안내** | 신규 출시 예적금·대출·카드·외환 상품 소개 |
| **금리 변동 안내** | 우대금리 인상, 한시 특판, 이벤트 금리 |
| **이벤트·캠페인** | 추첨 이벤트, 가입 사은품, 시즌 캠페인 |
| **우대 혜택 안내** | 본인 거래 실적 기반 우대조건 충족 안내 |
| **개인화 메시지** | 생일·기념일·결혼·입학 등 라이프 이벤트 |
| **제휴 상품 안내** | 카드사·보험사·증권사 제휴 상품 (별도 동의 시) |

(2) **콘텐츠 제공 빈도** (참고)

- 신상품 안내: 분기당 1~2회
- 금리 변동: 변동 발생 시
- 이벤트: 월 1~3회
- 개인화 메시지: 연 1~3회 (생일·기념일)

## 제3조 (수신 채널)

(1) **선택 가능한 채널**

고객은 다음 채널 중 일부 또는 전부를 선택할 수 있으며, 각 채널별로 개별 동의·철회가 가능합니다.

| 채널 | 발송 방식 | 비용 |
|---|---|---|
| SMS / LMS | 문자 메시지 | 무료 |
| MMS | 이미지·동영상 메시지 | 무료 |
| 이메일 | 등록 이메일 | 무료 |
| 모바일 앱 푸시 | 다온뱅크 앱 알림 | 무료 |
| 전화 (상담사 발신) | 영업시간 내 발신 | 무료 |
| 우편 (DM) | 등록 주소 | 무료 |

(2) **채널별 영업시간 제한**

| 채널 | 영업시간 제한 |
|---|---|
| SMS/LMS/MMS | 평일 09:00 ~ 20:30 (별도 동의 시 24시간) |
| 이메일 | 24시간 (시간 제한 없음) |
| 앱 푸시 | 평일 08:00 ~ 22:00 (별도 동의 시 24시간) |
| 전화 | 평일 09:00 ~ 20:00 (주말·공휴일 미발송) |
| 우편 | 영업일 기준 |

## 제4조 (보유 및 이용 기간)

(1) **동의 유효 기간**

- 동의일로부터 **2년** 유효
- 2년 경과 시 자동 종료
- 만료 30일 전 SMS·앱 알림으로 갱신 안내
- 갱신 미응답 시 자동 만료

(2) **동의 갱신 절차**

1. 만료 30일 전 알림 수신
2. 모바일 앱 또는 인터넷뱅킹에서 갱신 동의
3. 동의일로부터 2년 추가 연장
4. 미응답 시 만료, 마케팅 정보 자동 차단

(3) **만료 후 처리**

- 마케팅 정보 발송 즉시 중단
- 기존 보관 마케팅 활용 정보 (관심 상품, 거래 패턴) 익명화
- 본인 요청 시 즉시 삭제

## 제5조 (동의 철회)

(1) **철회 방법**

언제든 즉시 철회할 수 있으며, 다음 방법 중 하나를 선택할 수 있습니다.

- 모바일 앱: 「설정 → 알림 → 마케팅 동의 관리」
- 인터넷뱅킹: 「개인정보 관리 → 마케팅 동의 관리」
- 영업점 방문 또는 콜센터 1599-XXXX
- 수신 메시지 하단의 「수신거부」 링크 클릭
- 이메일 회신 「수신거부」 입력

(2) **철회 시 효력**

- 신청 즉시 처리 (실시간 반영)
- 처리 시점 이후 발송된 메시지부터 차단
- 채널별 개별 철회 가능 (예: SMS는 차단, 앱 푸시는 유지)
- 철회 후 익월부터 마케팅 정보 발송 완전 중단

(3) **재동의**

철회 후에도 본인이 원할 때 다시 동의할 수 있습니다. 재동의 시 동의일로부터 2년 유효합니다.

## 제6조 (광고성 정보의 표시)

(1) **법정 표시 의무**

모든 마케팅 정보에는 「정보통신망법」 제50조에 따라 다음 사항이 표시됩니다.

- 메시지 상단에 「(광고)」 표시
- 발신인 정보 (다온뱅크)
- 수신거부 방법 안내
- 무료 수신거부 전화번호

(2) **영업시간 외 발송 금지**

- 21:00 ~ 익일 08:00 사이 SMS·앱 푸시·전화 발송 금지
- 단, 본인이 「24시간 발송 동의」 별도 항목에 동의한 경우 예외

## 제7조 (마케팅 활용 정보)

(1) **수집·이용 항목**

마케팅 동의 시 다음 정보가 마케팅 활용을 위해 수집·분석됩니다.

| 정보 유형 | 활용 목적 |
|---|---|
| 거래 패턴 (이체·결제 빈도, 평균 잔액) | 맞춤 상품 추천 |
| 관심 상품 (앱 클릭 이력, 검색 기록) | 관심 분야 분석 |
| 보유 상품 (예적금·대출·카드) | 신규·교차 판매 |
| 라이프 이벤트 (결혼·출산·이사) | 개인화 메시지 |
| 위치 정보 (선택, 별도 동의) | 근거리 영업점·이벤트 안내 |

(2) **활용 기간**

마케팅 활용 정보는 본 동의 유효 기간(2년) 종료 후 즉시 익명화 또는 삭제됩니다.

## 제8조 (동의 거부 권리)

(1) **거부의 자유**

본 동의는 선택 사항이며, 동의하지 않을 권리가 보장됩니다.

(2) **거부 시 영향**

| 영향 | 설명 |
|---|---|
| 금융 서비스 이용 | 영향 없음 (예적금·대출·이체 모두 정상 이용) |
| 거래 안내 (이체 알림 등) | 영향 없음 (별개 동의로 운영) |
| 우대 혜택 가입 | 마케팅 채널을 통한 안내 미수신 (앱 내 「혜택 페이지」 에서 직접 확인 가능) |
| 이벤트 추첨 | 마케팅 채널을 통한 신청 안내 미수신 |

(3) **선택적 거부**

채널별·내용별로 부분 동의도 가능합니다. 예: SMS는 동의, 전화·우편은 거부.
