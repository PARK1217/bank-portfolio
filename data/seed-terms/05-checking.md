# 자유입출금예금 약관

본 약관은 다온뱅크(이하 "은행")이 취급하는 자유입출금예금(보통예금·당좌예금·마이너스통장 한도형 등)의 가입·운용·해지에 적용됩니다. 정기예금·적금 약관과 함께 수신 상품 약관 체계를 구성합니다.

## 제1조 (목적)

본 약관은 자유입출금예금 거래의 권리·의무를 명확히 정함으로써 고객의 일상 금융거래의 편의성과 안전성을 확보함을 목적으로 합니다.

## 제2조 (자유입출금예금의 정의 및 종류)

(1) **자유입출금예금**은 입금·출금의 시기와 금액에 제한 없이 자유롭게 이용 가능한 요구불 예금을 말합니다.

(2) 다온뱅크의 자유입출금예금 종류:

| 상품명 | 유형 | 특징 |
|---|---|---|
| 다온뱅크 자유입출금 통장 | 일반형 | 모든 고객, 기본형 |
| 급여이체 우대 통장 | 우대형 | 급여이체 등록 시 추가 우대금리 |
| 마이너스통장 한도형 | 한도여신형 | 약정 한도까지 음수 잔액 가능 |
| 외화 자유입출금 통장 | 외화형 | USD·EUR·JPY 등 외화 보관 |
| 법인 당좌예금 | 법인형 | 어음·수표 발행 가능 |

## 제3조 (가입 자격)

(1) **개인 가입**

- 만 14세 이상 (단, 어린이 통장은 친권자 대리 가능)
- 본인 명의 신분증 (주민등록증·운전면허증·여권)
- 본인 실명확인 필수

(2) **법인 가입**

- 법인등기부등본, 사업자등록증
- 인감증명서 및 사용인감계
- 대표자 신분증

(3) **외국인 가입**

- 외국인등록증 또는 여권
- 국내 체류 자격 및 주소 확인
- 외국환거래법상 비거주자 별도 절차

## 제4조 (입출금 거래)

(1) **입금 채널**

다음 모든 채널에서 입금이 가능합니다.

- 영업점 창구 (현금·수표·자기앞수표)
- ATM (자동입출금기, 현금·통장)
- 인터넷뱅킹 / 모바일 앱 (계좌이체)
- 자동이체 / 급여이체 / 펌뱅킹

(2) **출금 채널 및 본인 인증**

| 채널 | 본인 인증 방법 |
|---|---|
| 영업점 창구 | 통장 + 비밀번호 + 신분증 |
| ATM | 카드/통장 + 비밀번호 (+OTP) |
| 인터넷뱅킹 | 공동인증서 / 금융인증서 + OTP |
| 모바일 앱 | 간편비밀번호 / 생체인증 + OTP |
| 텔레뱅킹 | 자동응답 비밀번호 + OTP |

(3) **출금 가능 시간**

- 영업점: 평일 09:00 ~ 16:00
- ATM: 365일 24시간 (단, 24:00 ~ 00:30 점검 시간 제외)
- 인터넷뱅킹·모바일·텔레뱅킹: 365일 24시간

## 제5조 (이체 한도)

(1) **기본 한도** (이체 채널별)

| 채널 | 1회 한도 | 1일 한도 |
|---|---|---|
| 영업점 (창구 이체) | 5억원 | 10억원 |
| ATM | 600만원 | 1,500만원 |
| 인터넷뱅킹 / 모바일 (당행) | 5억원 | 5억원 |
| 인터넷뱅킹 / 모바일 (타행) | 1억원 | 3억원 |
| 텔레뱅킹 | 5,000만원 | 1억원 |

(2) **한도 변경**

본인 한도는 다음 절차로 변경 가능합니다.

1. 본인 인증 (생체인증·OTP 추가 인증)
2. 한도 변경 신청 (앱·인터넷뱅킹·영업점)
3. 7일간 신청 한도 적용 (점검 기간)
4. 정상 거래 확인 후 영구 적용

## 제6조 (수수료)

(1) **이체 수수료** (일반 고객 기준, 면제 우대 미적용)

| 거래 유형 | 수수료 |
|---|---|
| 당행 이체 | 면제 |
| 타행 이체 (영업시간 내) | 500원 |
| 타행 이체 (영업시간 외) | 700원 |
| 자동이체 등록 (당행) | 면제 |
| 자동이체 등록 (타행) | 회당 200원 |
| ATM 출금 (당행) | 면제 |
| ATM 출금 (타행) | 800원 ~ 1,200원 (시간대별) |

(2) **수수료 면제**

다음 조건 중 하나를 충족하는 고객은 모든 자동이체·타행이체 수수료가 면제됩니다.

- 급여이체 월 150만원 이상 등록
- 당행 신용대출 잔액 보유
- 정기예금/적금 1,000만원 이상 보유
- 주거래 등급 SILVER 이상
- 만 65세 이상 시니어 고객

(3) **기타 수수료**

- 통장 재발급: 2,000원
- 통장 분실 신고 후 재발급: 5,000원
- 인감 변경 / 비밀번호 재등록: 1,000원
- 외화 환전: 환율 우대표에 따름

## 제7조 (마이너스통장 한도형 특칙)

(1) **한도여신의 정의**

자유입출금예금 계좌에 약정 한도를 부여하여, 잔액이 0 이하로 떨어질 경우 약정 한도 범위까지 자동으로 대출이 실행되는 방식입니다.

(2) **이자 산정**

```
연체 시 매일의 음수 잔액 × 약정 대출금리 / 365 = 일 이자
월말에 누적 일 이자가 자동 출금
```

(3) **약정 한도**

| 신용등급 | 약정 한도 범위 |
|---|---|
| 1~2등급 | 5,000만원 ~ 2억원 |
| 3~4등급 | 3,000만원 ~ 1억원 |
| 5~6등급 | 1,500만원 ~ 5,000만원 |
| 7등급 이상 | 별도 심사 |

(4) **자동 해지 사유**

- 6개월 이상 한도 미사용 (인출 0원)
- 신용정보 등재
- 만기 미연장 후 30일 경과

## 제8조 (분실·도난 신고)

(1) **즉시 신고 의무**

통장·카드·비밀번호·OTP 분실 시 다음 채널로 즉시 신고하여야 합니다.

- 24시간 분실신고 콜센터: 1599-XXXX
- 모바일 앱 「긴급 분실신고」 메뉴
- 영업점 (영업시간 내)

(2) **신고 시 효력**

- 신고 접수 시점부터 해당 매체 즉시 이용 정지
- 신고 후 14일 이내 재발급 신청

(3) **부정 출금 시 책임 분담**

| 사유 | 책임 |
|---|---|
| 본인의 고의·중대한 과실 | 본인 책임 |
| 통장·카드 도난 후 신고 전 부정 출금 | 본인 50% / 은행 50% |
| 신고 후 부정 출금 | 은행 전액 책임 |
| 보이스피싱·스미싱 | 은행 50% + 보이스피싱 환급제도 |
| 시스템 오류 | 은행 전액 책임 |

(3) 책임 분담 결정은 「전자금융거래법」 시행령에 따르며, 최종 결정은 금융감독원 분쟁조정위원회를 통해 확정됩니다.

## 제9조 (이자)

(1) **이율**

자유입출금예금의 이율은 보통 연 0.10% ~ 1.50% 수준이며, 상품·잔액·우대조건에 따라 다음과 같이 차등 적용됩니다.

| 상품 | 기본 이율 | 우대 시 최고 |
|---|---|---|
| 일반 자유입출금 | 연 0.10% | 연 0.30% |
| 급여이체 우대 통장 | 연 1.50% | 연 3.50% |
| 외화 자유입출금 (USD) | 연 4.50% | 연 5.00% |

(2) **이자 지급 주기**

- 매 분기 마지막 영업일 (3월·6월·9월·12월 말)
- 일할 계산하여 분기말에 일괄 지급

(3) **세금**

이자소득세 14% + 지방소득세 1.4% = 15.4% 원천징수.

## 제10조 (휴면예금)

(1) 다음 조건에 모두 해당하는 계좌는 「휴면예금」으로 분류됩니다.

- 잔액 1만원 이하
- 5년 이상 입출금 거래 없음
- 등록된 연락처로 통지 불가

(2) 휴면예금은 「휴면예금관리재단」으로 출연되며, 출연 후에도 본인이 청구하면 즉시 반환됩니다.

(3) 휴면예금 출연 6개월 전 등록된 모든 연락처로 안내 발송됩니다.

## 제11조 (예금자 보호)

(1) 본 자유입출금예금은 「예금자보호법」에 따라 예금보험공사가 보호합니다.

(2) **보호 한도**: 1인당 5,000만원 (다온뱅크의 다른 예적금 합산, 원금 + 이자 포함)

(3) **외화예금**: 원화 환산 기준으로 보호 한도가 적용됩니다.

## 제12조 (분쟁 처리)

(1) 자유입출금예금 관련 분쟁은 다음 순서로 처리합니다.

1. 영업점 또는 24시간 콜센터(1599-XXXX) 신청
2. 은행 민원처리부 정식 접수
3. 금융감독원 금융분쟁조정위원회
4. 한국소비자원 분쟁조정
5. 법원 조정·소송

(2) 대표 분쟁 사례

- 부정 출금 (보이스피싱·해킹): 「전자금융거래법」 책임 분담 적용
- 수수료 면제 미적용: 우대조건 충족 증빙 후 환급
- 한도 변경 미반영: 시스템 점검 후 소급 적용
- 휴면예금 무단 전환: 본인 청구 시 즉시 반환

## 부칙

본 약관은 2026년 1월 1일부터 시행합니다.
