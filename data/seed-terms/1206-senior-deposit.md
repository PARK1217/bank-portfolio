# 시니어 정기예금 상품 특약약관

본 특약약관은 「시니어 정기예금」(상품번호 206, **만 60세 이상 우대 상품**)에 적용되는 상품 특별 조항입니다.

## 제1조 (상품 개요)

| 항목 | 내용 |
|---|---|
| 상품명 | 시니어 정기예금 |
| 상품 유형 | 시니어 우대 거치식 정기예금 |
| 가입 대상 | **만 60세 이상** 개인 |
| 가입 금액 | 100만원 ~ 5억원 |
| 가입 기간 | 12 / 24 / 36개월 (3종) |
| 기본 이율 | 연 3.75% (12개월) / 3.85% (24개월) / 4.00% (36개월) |
| 최고 우대 이율 | 연 4.50% |
| 가입 채널 | 모바일 앱 / 인터넷뱅킹 / 영업점 |

## 제2조 (가입 자격)

(1) **연령 자격**

- **만 60세 이상** 거주자 (가입 시점 만 60세 생일 도래 필요)
- 미만 연령은 가입 불가 (만 60세 미만 고객은 「행복드림 정기예금」(201) 권장)

(2) **자격 증빙**

- 주민등록증 또는 운전면허증으로 자동 확인
- 외국인은 외국인등록증 + 국적·생년월일 확인

(3) **1인 가입 한도**

- 1인 최대 5계좌 가입 가능
- 합산 가입금액 한도 없음

## 제3조 (기간별 이율)

| 가입 기간 | 기본 이율 (연) | 우대 충족 시 |
|---|---|---|
| 12개월 | 3.75% | 4.25% |
| 24개월 | 3.85% | 4.35% |
| 36개월 | 4.00% | 4.50% |

## 제4조 (우대 이율 조건)

(1) **시니어 전용 우대 항목**

| 우대 항목 | 우대 폭 |
|---|---|
| **만 60세 이상 (기본 적용)** | +0.30%p |
| **국민연금·기초연금 수령 실적** (월 30만원 이상) | +0.20%p |

(2) **최고 이율 산정 예시 (36개월, 연금 수령 시)**

```
기본 이율 (36개월)  : 연 4.00%
+ 시니어 우대       : +0.30%p
+ 연금수령 우대     : +0.20%p
────────────────────
합계               : 연 4.50%
```

## 제5조 (이자 산정 및 지급)

(1) **단리 적용** (정기예금 표준)

```
세전 이자 = 원금 × 적용 이율 × (예치 일수 / 365)
```

(2) **이자 지급 방식**

| 방식 | 시니어 권장 활용 |
|---|---|
| 만기일시지급 | 만기에 일괄 수령 |
| **월이자지급 (시니어 권장)** | 매월 이자를 생활비로 사용 |
| 분기이자지급 | 3개월마다 지급 |

(3) **시뮬레이션 (5,000만원 × 36개월 × 우대 4.50%, 월이자지급)**

```
연간 세전 이자 = 50,000,000 × 0.045 = 2,250,000원
월간 세전 이자 = 2,250,000 / 12 ≈ 187,500원
월간 세후 이자 ≈ 158,625원 (세금 15.4% 차감)
36개월 누적 세후 이자 ≈ 5,710,500원
```

(4) **비과세종합저축 (만 65세 이상)**

만 65세 이상 고객은 「비과세종합저축」 한도(1인 5,000만원) 내에서 이자소득세가 면제됩니다. 가입 시 「비과세종합저축 가입신청서」 별도 작성.

## 제6조 (이자 지급 방식 변경)

(1) 가입 후 이자 지급 방식은 1회 변경 가능합니다.

- 만기일시지급 → 월이자지급 (또는 분기이자지급)
- 월이자지급 → 만기일시지급 (적용된 이자는 재예치)

(2) 변경 신청은 영업점·앱·인터넷뱅킹에서 가능합니다.

## 제7조 (중도해지)

| 경과 비율 | 적용 이율 |
|---|---|
| 1개월 미만 | 약정금리의 10% |
| 1~3개월 | 약정금리의 30% |
| 3~6개월 | 약정금리의 40% |
| 6~9개월 | 약정금리의 55% |
| 9~12개월 | 약정금리의 70% |
| 만기 80% 이상 | 약정금리의 90% |

(1) 중도해지 시 모든 우대금리는 회수됩니다.

(2) **시니어 우대 특례**

만 75세 이상 고객의 부득이한 사유(의료비·요양비)에 의한 중도해지는 약정금리의 100%가 적용됩니다 (위약 없음).

## 제8조 (만기 처리)

(1) **만기 자동 처리**

- **만기 자동지급**: 만기일에 원금+이자 지정 계좌
- **만기 자동연장**: 동일 기간으로 재가입 (만기 시점 이율 적용)
- **자녀 명의 자동 입금**: 사전 등록한 자녀 계좌로 직접 입금 (상속·증여 사전 설계)

(2) **만기 후 미해지**

만기 후 자동 해지가 이루어지지 않은 경우 약정금리의 50%가 적용됩니다 (1개월 이내).

## 제9조 (시니어 전용 추가 혜택)

(1) **영업점 우선 응대**

영업점 방문 시 「시니어 전용 창구」 우선 응대 (대기 시간 단축).

(2) **콜센터 우선 연결**

1599-XXXX 콜센터에서 만 65세 이상 자동 우선 연결.

(3) **앱 큰글씨 모드**

다온뱅크 모바일 앱에서 「큰글씨 모드」 자동 활성화 (1.5배 확대).

(4) **금융 사고 예방 안내**

보이스피싱·금융사기 예방 안내가 분기 1회 발송됩니다.

## 제10조 (예금자 보호)

「예금자보호법」 보호 대상. 1인당 5,000만원 한도.

## 제11조 (상속 설계 안내)

(1) **사전 등록 가능 항목**

- 만기 시 자녀 명의 자동 입금 계좌
- 본인 사망 시 상속인 정보 (사전 등록 시 절차 간소화)

(2) **본인 사망 시 처리**

- 사망 신고 즉시 본 정기예금 일시 정지
- 상속인 청구 후 약정금리의 100% 적용 (위약 없음)
- 상속 비율에 따라 분할 지급 가능

## 부칙

본 특약은 2026년 1월 1일부터 시행합니다.
