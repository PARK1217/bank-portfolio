# 급여이체 우대 통장 상품 특약약관

본 특약약관은 「급여이체 우대 통장」(상품번호 102) 에 적용되는 상품 특별 조항입니다. 본 특약에 정함이 없는 사항은 「자유입출금예금 약관」(TERMS_ID 5)에 따릅니다.

## 제1조 (상품 개요)

| 항목 | 내용 |
|---|---|
| 상품명 | 급여이체 우대 통장 |
| 상품 유형 | 자유입출금예금 + 급여이체 연계 우대 |
| 가입 대상 | 만 19세 이상 직장인 (재직 6개월 이상) |
| 가입 금액 | 1원 이상 |
| 가입 기간 | 무기한 |
| 기본 이율 | 연 1.50% |
| 최고 우대 이율 | 연 3.50% (모든 우대 충족 시) |
| 잔액 한도 | 우대 이율은 1,500만원까지 적용 (초과분은 기본 이율) |

## 제2조 (가입 자격)

(1) **개인 가입**

- 만 19세 이상
- 재직 6개월 이상 (재직증명서·근로계약서 제출)
- 본인 명의 다른 금융기관의 급여이체를 다온뱅크로 변경 가능

(2) **재직 증빙 서류**

- 재직증명서 (3개월 이내 발급)
- 근로계약서
- 4대보험 가입증명서 (선택)
- 1인당 1계좌 한정

## 제3조 (우대 이율 조건)

(1) **우대 항목 및 우대폭**

| 우대 항목 | 우대 폭 | 평가 주기 |
|---|---|---|
| **급여이체 등록 + 월 150만원 이상 입금** | +1.40%p | 월 평가 |
| **다온뱅크 체크카드 월 30만원 이상 사용** | +0.30%p | 월 평가 |
| **자동이체 3건 이상 등록** | +0.20%p | 월 평가 |
| **마이데이터 연동** | +0.10%p | 가입 시 1회 |

(2) **최고 우대 이율 산정 예시**

```
기본 이율           : 연 1.50%
+ 급여이체 우대      : +1.40%p
+ 체크카드 우대      : +0.30%p
+ 자동이체 우대      : +0.20%p
+ 마이데이터 우대    : +0.10%p
────────────────────
합계               : 연 3.50%
```

## 제4조 (우대 적용 방식)

(1) **월 평가**: 매월 말일 기준 우대 조건 충족 여부를 평가하여 다음 달 이율에 반영

(2) **일할 계산**: 우대 이율은 충족된 달의 일수에 대해서만 적용

(3) **잔액 한도 초과분 처리**: 1,500만원 초과 잔액은 기본 이율 (연 1.50%) 만 적용

(4) **예시**

```
잔액 1,500만원, 1개월 (30일) 보유, 우대 조건 모두 충족:
세전 이자 = 15,000,000 × 0.035 × 30 / 365 ≈ 43,150원
세후 이자 ≈ 36,505원

잔액 3,000만원, 동일 조건:
1,500만원 부분: 우대 이율 3.50%
초과 1,500만원: 기본 이율 1.50%
합산 세전 이자 ≈ 43,150 + 18,493 = 61,643원
```

## 제5조 (급여이체의 정의)

(1) 다음 입금이 「급여이체」로 인정됩니다.

- 사업자 회사 코드를 통한 급여 입금 (펌뱅킹)
- 인터넷뱅킹 또는 모바일에서 「급여」 사유로 입금
- 매월 정기 (10일 이내 변동) + 동일 입금자 + 150만원 이상

(2) **인정 불가**

- 본인 명의 다른 계좌에서의 자기 이체
- 가족·지인의 단순 이체
- 「급여」 사유 외 사유 (생활비·용돈·환급 등)

## 제6조 (수수료 면제)

본 상품 가입 후 급여이체 등록 시 다음 수수료가 모두 면제됩니다.

- 당행 타행 이체 (모든 채널)
- 자동이체 등록 (당행·타행)
- ATM 타행 출금 (월 10회 한도)
- 통장 재발급, 인감 변경 등 부가 수수료

## 제7조 (해지 및 우대 종료)

(1) **상품 해지**: 즉시 가능, 잔액은 지정 계좌로 입금

(2) **우대 종료 사유**

- 6개월 연속 급여이체 미입금
- 본인 퇴직 (재직증명서 갱신 불가)
- 다른 금융기관으로 급여이체 변경

우대 종료 시 본 상품은 자동으로 「다온뱅크 자유입출금 통장」(상품 101)과 동일한 이율(연 0.10%)로 전환됩니다.

## 제8조 (예금자 보호)

「예금자보호법」에 따라 예금보험공사가 보호합니다. 보호 한도 1인당 5,000만원.

## 부칙

본 특약은 2026년 1월 1일부터 시행합니다.
