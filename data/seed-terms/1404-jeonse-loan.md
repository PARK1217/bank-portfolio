# 전세자금대출 상품 특약약관

본 특약약관은 「전세자금대출」(상품번호 404)에 적용되는 상품 특별 조항입니다.

## 제1조 (상품 개요)

| 항목 | 내용 |
|---|---|
| 상품명 | 전세자금대출 |
| 상품 유형 | 보증부 가계대출 (전세보증금 자금) |
| 가입 대상 | 만 19세 이상 무주택 세입자 |
| 대출 한도 | **임차보증금의 80%** (수도권) / 70% (비수도권) |
| 대출 기간 | 12 / 24개월 (전세 계약 기간 연동) |
| 기본 금리 | 연 4.20% (12개월) / 4.40% (24개월) |
| 최저 우대 금리 | **연 3.30%** |
| 보증기관 | 한국주택금융공사 / 서울보증보험 |

## 제2조 (가입 자격)

(1) **차주 자격**

- 만 19세 이상 ~ 만 65세 미만
- 무주택 세대주 (배우자 명의 주택 포함, 단 임대 중인 경우 인정)
- 다온뱅크 자유입출금예금 계좌 보유
- 연소득 1,800만원 이상 (저소득 청년·신혼부부 별도)

(2) **임대차계약 자격**

- 임차보증금 한도 4억원 (수도권) / 2억원 (비수도권)
- 임대인 동의 (전세권 설정 또는 확정일자)
- 잔금 지급 5영업일 전 신청 가능

(3) **주택 자격**

- 임차 주택 등기 완료
- 임대인 명의 부합 (등기부등본 확인)
- 사회복지시설·연립주택은 별도 심사

## 제3조 (대출 한도)

(1) **임차보증금 대비**

| 지역 | LTV (보증금 대비) |
|---|---|
| 수도권 (서울·경기·인천) | 80% |
| 비수도권 | 70% |

(2) **DSR 한도**: 40%

(3) **시뮬레이션 (수도권, 임차보증금 3억원)**

```
최대 한도 (80%): 2억 4천만원
- 본인 자기자금: 6,000만원
- 다온뱅크 전세자금대출: 2억 4천만원

월 이자 (24개월 × 4.40%): 약 88만원
- 연소득 5,000만원, DSR 40% → 연 2,000만원 한도 OK
```

## 제4조 (기간별 금리)

| 대출 기간 | 기본 금리 (연) | 최저 우대 (연) |
|---|---|---|
| 12개월 | 4.20% | 3.30% |
| 24개월 | 4.40% | 3.50% |

## 제5조 (우대 금리)

| 우대 항목 | 우대 폭 |
|---|---|
| **신혼부부 (혼인 7년 이내)** | -0.50%p |
| **만 34세 이하 청년** | -0.40%p |

> ⚠ **신혼부부 + 청년 우대는 중복 적용 불가**. 더 유리한 1개만 적용.

## 제6조 (상환 방식)

(1) **만기일시상환** (기본)

매월 이자만 납부, 만기일에 원금 일시 상환.

**예시 (전세자금 2억원 × 24개월 × 3.50% 우대)**

```
월 이자 = 200,000,000 × 0.035 / 12 ≈ 583,333원
24개월 누적 이자: 약 1,400만원
만기 원금 일시 상환: 2억원
```

(2) **원리금균등** (선택, 일부 상품)

매월 동일한 원리금 상환. 자녀 출산·승진 등으로 소득 증대 예상 시 권장.

## 제7조 (보증료)

| 보증기관 | 보증료율 (연) | 비고 |
|---|---|---|
| 한국주택금융공사 | 0.30% ~ 0.50% | 신혼부부·청년 우대 시 0.10%p 추가 우대 |
| 서울보증보험 | 0.50% ~ 0.80% | 한도 초과 시 |
| 신용보증재단 | 0.40% | 정책성 |

보증료는 매년 1회 일괄 납부 (잔액 기준).

## 제8조 (실행 절차)

1. 임대차계약서 + 임대인 동의서 + 본인 신분증 준비
2. 모바일 앱 「대출 한도 조회」 (가심사)
3. 본 심사 (재직·소득·신용평가)
4. 보증기관 보증서 발급
5. 약정 (전자약정서)
6. **잔금 지급일에 임대인 계좌로 직접 입금** (본인 통장 경유 X)

> ⚠ **본인 명의 통장 입금 후 임대인 송금 방식 불가**. 본 상품은 임대인 계좌 직송금이 원칙.

## 제9조 (전세계약 갱신)

(1) **자동 연장 (24개월 → 48개월)**

전세 계약을 갱신한 경우 본 대출도 갱신 가능.

(2) **갱신 절차**

- 갱신 1개월 전 SMS·앱 알림
- 갱신된 임대차계약서 제출
- 신용평가 재실시
- 갱신 약정 체결

(3) **갱신 거절 사유**

- 신용 상태 악화
- 임대인 변경 (새 임대인 동의 필요)
- 임차보증금 인상 시 추가 자기자금 필요

## 제10조 (계약 종료 시 처리)

(1) **임차 계약 정상 종료**

임차보증금 반환 시점에 임대인이 본 대출 잔액을 다온뱅크로 직접 송금.

(2) **임대인 보증금 반환 지연**

- 14일 이내 미반환 시 보증기관에 보증 청구
- 본인의 대출 잔액은 다온뱅크 잔여 잔액 한도까지 보증

## 제11조 (중도상환 수수료)

| 약정 후 경과기간 | 수수료율 |
|---|---|
| 1년 이내 | 0.5% |
| 1년 초과 | 면제 |

## 제12조 (예금자 보호)

본 상품은 「대출」이므로 보호 대상 아님. 단, 임차보증금 자체는 「주택임대차보호법」 보호 대상.

## 부칙

본 특약은 2026년 1월 1일부터 시행합니다.
