# 프리미엄 거치식 정기예금 상품 특약약관

본 특약약관은 「프리미엄 거치식 정기예금」(상품번호 202, **특판 상품**)에 적용되는 상품 특별 조항입니다.

## 제1조 (상품 개요)

| 항목 | 내용 |
|---|---|
| 상품명 | 프리미엄 거치식 정기예금 |
| 상품 유형 | 고액 거치식 정기예금 (단리) |
| 가입 대상 | 만 19세 이상 개인 / 법인 |
| 가입 금액 | **1,000만원 이상** ~ 50억원 |
| 가입 기간 | 12 / 24개월 (2종) |
| 기본 이율 | 연 4.00% (12개월) / 연 4.20% (24개월) |
| 최고 우대 이율 | 연 4.30% (모든 우대 충족 시) |
| 판매 한도 | 한시 특판 (총 500억원 한도, 소진 시 종료) |

## 제2조 (특판 안내)

(1) **판매 기간**: 2026년 1월 1일 ~ 2026년 6월 30일 (6개월 한정)

(2) **판매 한도**: 총 500억원 (선착순 마감)

(3) **소진 안내**: 판매 한도 소진 시 즉시 판매 종료. 가입 신청 후 한도 부족으로 가입 불가 시 신청 취소.

## 제3조 (가입 자격)

- 만 19세 이상 (미성년자 가입 불가)
- 본인 명의 다온뱅크 자유입출금예금 계좌
- 자금 출처 증빙 (1억원 이상 가입 시)

## 제4조 (기간별 적용 이율)

| 가입 기간 | 기본 이율 (연) | 우대 충족 시 최고 (연) |
|---|---|---|
| 12개월 | 4.00% | 4.30% |
| 24개월 | 4.20% | 4.50% |

## 제5조 (우대 이율 조건)

(1) **우대 항목**

| 우대 항목 | 우대 폭 | 비고 |
|---|---|---|
| **가입금액 1억원 이상** | +0.20%p | 가입 시 즉시 |
| **만기 자동연장 신청** | +0.10%p | 가입 시 신청 |

(2) **최고 이율 산정 예시 (1억원 × 24개월)**

```
기본 이율           : 연 4.20%
+ 가입금액 우대     : +0.20%p
+ 만기연장 우대     : +0.10%p
────────────────────
합계               : 연 4.50%
```

## 제6조 (이자 산정 시뮬레이션)

(1) **5,000만원 × 12개월 × 우대 4.10%**

```
세전 이자 = 50,000,000 × 0.041 = 2,050,000원
세금 (15.4%) = 315,700원
세후 이자  = 1,734,300원
만기 수령액 = 51,734,300원
```

(2) **1억원 × 24개월 × 우대 4.50%**

```
세전 이자 = 100,000,000 × 0.045 × 2 = 9,000,000원
세금 (15.4%) = 1,386,000원
세후 이자  = 7,614,000원
만기 수령액 = 107,614,000원
```

(3) **5억원 × 24개월 × 우대 4.50%**

```
세전 이자 = 500,000,000 × 0.045 × 2 = 45,000,000원
세금 (15.4%) = 6,930,000원
세후 이자  = 38,070,000원
만기 수령액 = 538,070,000원
```

## 제7조 (이자 지급 방식)

| 방식 | 설명 |
|---|---|
| 만기일시지급 | 만기일에 원금+이자 일괄 (기본) |
| 월이자지급 | 매월 이자 지정 계좌 지급 |
| 분기이자지급 | 3개월마다 지급 |

> ⚠ **만기자동재예치는 본 상품 특판 종료 시 일반 정기예금(상품 201)으로 자동 전환됩니다.**

## 제8조 (중도해지)

| 경과 비율 | 적용 이율 |
|---|---|
| 1개월 미만 | 약정금리의 10% |
| 1~3개월 | 약정금리의 30% |
| 3~6개월 | 약정금리의 40% |
| 6~9개월 | 약정금리의 55% |
| 9개월~만기 90% | 약정금리의 70% |
| 만기 90% 이상 | 약정금리의 90% |

중도해지 시 모든 우대금리는 회수되며, 「기본금리 기준」으로 적용됩니다.

## 제9조 (만기 처리)

(1) **만기 자동 처리 (특판 기간 종료 후)**

본 상품은 특판 종료 후 만기 자동연장 시 다음 상품으로 자동 전환됩니다.

- 「행복드림 정기예금」(상품 201) 동일 기간으로 재가입
- 재가입 시 「행복드림 정기예금」의 기본 이율과 우대조건이 적용됩니다

(2) **만기 후 이율 (자동 처리 미신청 시)**

| 경과기간 | 적용 이율 |
|---|---|
| 1개월 이내 | 약정금리의 50% |
| 1~3개월 | 약정금리의 30% |
| 3개월 초과 | 연 0.10% |

## 제10조 (예금자 보호)

「예금자보호법」 보호 대상. 1인당 5,000만원 한도. 1억원 이상 가입 시 보호 한도 초과분 안내가 가입 시 제공됩니다.

## 제11조 (자금 출처 증빙 (1억원 이상))

(1) 1억원 이상 가입 시 다음 서류 중 하나가 요구됩니다.

- 부동산 매매계약서 / 거래내역
- 퇴직금 수령 증빙
- 보험 만기 환급 증빙
- 사업소득 증빙 (사업자등록증 + 소득금액증명원)
- 상속·증여 증빙

(2) 「특정금융정보의 보고 및 이용 등에 관한 법률(AML)」에 따라 자금세탁방지 점검이 이루어집니다.

## 부칙

본 특약은 2026년 1월 1일부터 시행하며, 판매 기간은 2026년 6월 30일까지 또는 한도 소진 시까지입니다.
